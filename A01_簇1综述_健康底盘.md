# 母题簇综述｜簇1：健康底盘与生物优化

> 内含5张卡：C1-1多巴胺 / C1-2饮食 / C1-3节律 / C1-4成瘾 / C1-5睡眠
> 跨簇连接：簇6执行 / 簇2认知 / 簇3财富

---

## 核心论点

**健康不是你"做得好"的事情之一——它是所有其他事情能做的前提。** 如果健康底盘崩了，认知、执行、创作、财富、关系全部跟着崩。健康的优先级应高于一切其他目标，因为它是所有其他目标的"乘性因子"——健康=0时，其他所有×0=0。

## 五张卡的逻辑链

1. **C1-1多巴胺基线崩塌**：诊断执行力的底层生理限制——低多巴胺时，任何需要意志力的行为都变成逆流游泳。
2. **C1-4成瘾不是意志力问题**：深化诊断——大脑没有替代品时自动寻求最高刺激源。替代疗法>戒断疗法。
3. **C1-5睡眠不是休息**：熬夜真正原因不是手机好玩，是白天没给自己足够成就感。睡眠是系统健康度最敏感指标。
4. **C1-3昼夜节律重建**：提供"晨光锚定"免费重置工具——先确定你是早鸟还是夜猫，再定入睡时间。
5. **C1-2饮食决策框架**：每天做最多次的决策必须框架化——靠提前准备好的默认选项，而非临时判断。

## 跨簇洞察

- **→ 簇6**：健康是P0单一主线的唯一正确选择。健康稳定后执行系统才能运转。
- **→ 簇2**：多巴胺模型直接解释认知目标函数U值波动——低多巴胺=低认知。
- **→ 簇3**：健康复利前2周看不出变化，第30天突然发现"看书有意思"。

## 对你当前的建议

把"恢复健康"翻译成3个可追踪数字：每天睡眠>7.5h + 每周运动3次 + 每天精制糖<15g。Google Sheet追踪30秒/天。连续4周完成率>70%=底盘稳定。在此之前，不开启任何新P0项目。
