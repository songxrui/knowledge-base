# 内容发布后数据追踪框架

> 创建时间: 2026-06-05
> 来源: dbs-chatroom判官建议 — "缺少量化反馈闭环"
> 用途: 每篇内容发布后填写，积累数据驱动内容迭代

---

## 追踪维度

### 核心指标（每篇必填）

| 指标 | 定义 | 记录方式 |
|---|---|---|
| 发布时间 | YYYY-MM-DD HH:MM | 自动 |
| 平台 | 小红书/公众号/抖音/X | 选择 |
| 内容类型 | 长文/短内容/视频/图文 | 选择 |
| 阅读量/播放量 | 72小时数据 | 手动 |
| 互动率 | (点赞+收藏+评论)/阅读量 | 计算 |
| 转发/分享数 | 跨平台传播指标 | 手动 |
| 转化咨询数 | 由此内容带来的私信/咨询 | 手动 |
| 转化付费数 | 由此内容带来的付费 | 手动 |

### 质量指标（每周汇总）

| 指标 | 计算方式 |
|---|---|
| 平均阅读量 | 当周所有内容阅读量平均 |
| 爆款率 | 阅读量>平均值2倍的内容占比 |
| 互动率趋势 | 每周互动率变化 |
| 转化率 | 咨询数/阅读量 |
| 内容寿命 | 发布后持续获得流量的天数 |

### 内容质量自评（每篇）

| 维度 | 1-5分 |
|---|---|
| 信息密度 | 读者能带走多少可用的东西 |
| 情感共鸣 | 读者有多大程度"被触动" |
| 行动触发 | 读者读完后想不想做点什么 |
| 差异化 | 和其他同类内容的区别度 |
| 可复用性 | 6个月后是否仍有价值 |

---

## 数据驱动迭代规则

| 信号 | 行动 |
|---|---|
| 阅读量高但互动率低 | 标题/封面好，但内容不够"有共鸣"→加强情感锚点 |
| 互动率高但转化低 | 内容好但缺少行动引导→加强CTA(行动呼吁) |
| 转化高但阅读量低 | 精准但受众窄→考虑扩大选题范围或保持垂直深耕 |
| 所有指标都低 | 选题/标题/内容都需要重新审视 |
| 内容寿命长(>30天仍有流量) | 这是你的"长期资产"——深度内容方向正确 |

```
# {月份}内容数据复盘

- 总阅读: {N}
- 平均互动率: {X}%
- 转化咨询: {N}
- 转化付费: {N}

## 最佳表现内容
1. {标题} — 阅读{N} / 互动{X}% / 转化{N}
   - 成功因素: {分析}

## 最差表现内容

   - 改进方向: {分析}

## 内容资产累积
- 本月新增内容单元: {N}
- 本月新增装配稿: {N}
- 累计可复用资产: {N}单元/{N}装配稿

## 下月策略调整
- {基于数据的3条调整}
```
