# 爱是能力，不是感觉——亲密关系的底层逻辑

> （信源待补充）

---

"我不爱他了"——这是结束关系最常见的理由。但这句话的错误在于，它把爱当成了一种感觉，而不是一种能力。


## 感觉vs能力

| 爱作为感觉 | 爱作为能力 |
|----------|----------|
| 发生了，你无法控制 | 你选择去做 |
| 可能明天就消失 | 可以持续练习和提升 |
| 不用努力 | 需要刻意练习 |
| "我坠入爱河" | "我选择去爱" |

[推断] 把爱当感觉的人，每段关系都死于"感觉淡了"。把爱当能力的人，知道感觉起伏是正常的，能力才是关系的底盘。


## 爱的五种核心能力

1. **注意的能力**：真正看见对方，不是看着手机点头
2. **倾听的能力**：听对方在说什么，而不是在想自己怎么回应
3. **表达的能力**：把自己的感受说清楚，不是让对方猜
4. **修复的能力**：发生冲突后，谁能先伸出橄榄枝
5. **放手的能能力**：知道什么时候该尊重对方的独立空间


## 长期关系的真相

[信源支持] 研究表明：所有长期关系都会经历激情消退的阶段（通常18-36个月后）。区别在于：
- 把爱当感觉的人：激情消退 = "我不爱了" → 分手
- 把爱当能力的人：激情消退 = "进入下一阶段了" → 转向陪伴型亲密

不会游泳的人换游泳池没用。不会爱的人换伴侣也没用。


## 行动

1. 五能力自评：你最强的和最弱的是哪个？
2. 本周刻意练习最弱的那项能力
3. 下次产生"我不爱了"的念头时，问自己：是感觉淡了，还是我没在练习？
