# 封面图生成指南

基于 baoyu-cover-image skill 方法论，为知识库文章生成社交分享封面图。

## 设计规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 1200×630px (公众号/推特) |
| 配色 | 深色背景(#1a1a2e) + 蓝色强调(#16213e → #0f3460) |
| 字体 | 标题: 粗黑体 48px / 副标题: 常规 28px |
| 布局 | 标题居中 + 底部作者署名 |

## 封面模板结构

`
┌──────────────────────────────────┐
│                                  │
│         [核心概念图标]            │
│                                  │
│    标题（12-18字，加粗）          │
│                                  │
│    「一句话引用/金句」            │
│                                  │
│              khazix 知识库        │
└──────────────────────────────────┘
`
