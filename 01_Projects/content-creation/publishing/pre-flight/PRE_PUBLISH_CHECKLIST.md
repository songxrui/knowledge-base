# 发布媒体前自检笔记

> *"流量的本质，是人类注意力的时间切片。" — Stanley.team*
> 
> *发布前逐项检查。不是每一项都要满分，但每一项你都必须知道自己打了多少分。*

## 一、注意力第一性检查（5秒测试）

| # | 检查项 | 标准 | 你的内容是否通过？ |
|---|--------|------|-------------------|
| 1 | **3秒钩子** | 前3秒/前3行有认知冲突或信息缺口？ | ☐ |
| 2 | **情绪锚点** | 内容触发了敬畏/愤怒/焦虑/幽默中的至少一种？ | ☐ |
| 3 | **一句话核心** | 陌生人能用一句话说清"这篇在讲什么"吗？ | ☐ |
| 4 | **视觉第一印象** | 封面/缩略图在信息流中能让人停下拇指滑动？ | ☐ |
| 5 | **能耗判断** | 用户"理解这篇"需要消耗多少注意力？越低越好 | ☐ |

> 引用来源：传播学5W模型-效果层、2023-2025 短视频注意力研究"前3秒决定70%完播率"

## 二、心理学触发器检查

| # | 偏误/触发 | 应用方式 | 已使用？ |
|---|----------|---------|---------|
| 1 | **好奇心缺口** | 标题埋悬念，不给完整答案，内容中段再开新坑 | ☐ |
| 2 | **损失厌恶** | 有没有"如果不做X会损失Y"的表达？ | ☐ |
| 3 | **社会认同** | 有没有"X万人已收藏/已实践"类的数字证明？ | ☐ |
| 4 | **峰终定律** | 中间可平，但峰值时刻和结尾是否做到了极致？ | ☐ |
| 5 | **框架效应** | 同样的信息，表述方式选择了"得"框架还是"失"框架？ | ☐ |
| 6 | **稀缺效应** | 有没有限时/限量/独家感？ | ☐ |
| 7 | **权威偏误** | 有没有引用数据源/专家/研究作为背书？ | ☐ |
| 8 | **信息差焦虑** | 有没有"大多数人不知道的X"结构？ | ☐ |
| 9 | **禀赋效应** | 有没有让读者产生"这是我的内容"的感觉？ | ☐ |

**情绪能量评级**：□ 敬畏 □ 愤怒 □ 焦虑 □ 幽默（高传播） / □ 悲伤 □ 平静 □ 满足（低传播）

> 引用来源：心理学武器库21机制、系统1/2框架(Kahneman, 2011)、Berger & Milkman (2012) 病毒传播研究

## 三、分享动机核实

每一条分享背后都有一个心理动机。你的内容触发了几条？

| # | 分享动机 | 一句话验证 | 触发？ |
|---|---------|----------|-------|
| 1 | **社会货币** | 转发这篇会让我在朋友圈显得更聪明/有品味吗？ | ☐ |
| 2 | **实用价值** | 这篇内容对我的朋友真的有用吗？ | ☐ |
| 3 | **情绪价值** | 内容是否引发了强烈情绪需要"倾倒"给他人？ | ☐ |
| 4 | **触发器** | 内容是否关联了某个高频日常场景？（如"每次开会前"） | ☐ |
| 5 | **公开可见** | 分享这个行为本身是否是可见的社交信号？ | ☐ |

> 引用来源：传播学病毒扩散模型(Bass, 1969)、分享动机研究

## 四、平台适配检查

### 小红书 (图文+搜索驱动)

| # | 检查项 | 通过？ |
|---|--------|-------|
| 1 | 封面图是否在信息流中有视觉冲击力？（大字+高对比色） | ☐ |
| 2 | 标题是否包含可搜索关键词？（用户会搜的词） | ☐ |
| 3 | 正文是否有"种草感"——真实体验而非说教？ | ☐ |
| 4 | 是否使用了平台标签（#）但不堆砌（3-5个精准标签）？ | ☐ |
| 5 | 是否有收藏价值？（清单/教程/模板类优先） | ☐ |

### 抖音 (完播率+互动率双驱动)

| # | 检查项 | 通过？ |
|---|--------|-------|
| 1 | 前3秒是否制造了"不可跳过"的认知冲突？ | ☐ |
| 2 | 全片是否有至少2个"峰"——让人情绪起伏的点？ | ☐ |
| 3 | 结尾是否有互动引导（评论/点赞/关注的具体理由）？ | ☐ |
| 4 | 内容是否有情绪化表达？（温和中立=小众） | ☐ |

| # | 检查项 | 通过？ |
|---|--------|-------|
| 1 | 标题是否同时满足"可搜索"和"可传播"两个要求？ | ☐ |
| 2 | 正文是否有≥2处"金句"——可被截图转发的句子？ | ☐ |
| 3 | 排版是否有呼吸感？（段落短、留白多、重点加粗） | ☐ |
| 4 | 结尾是否有明确的"下一步行动"引导？ | ☐ |

## 五、传播学模型校验

### 5W 完整度检查

| W | 问题 | 你的答案 |
|---|------|---------|
| **Who** | 谁在说？（你的身份/人设是什么） | |
| **What** | 说了什么？（一句话核心观点） | |
| **Channel** | 通过什么渠道？（平台=抖音/小红书/公众号/X） | |
| **Whom** | 对谁说？（目标受众的精准画像） | |
| **Effect** | 期望产生什么效果？（认知改变/行动/情绪/分享） | |

### 议程设置检查
- 这篇内容是否在"设定问题"而不仅仅是"给出答案"？ ☐
- 用户看完后，是否会开始用你的词汇和框架理解这个领域？ ☐

### 两级传播检查
- 这篇内容是否适合被"意见领袖"转发？ ☐
- 有没有可能触达的KOL/KOC可以帮你扩散？ ☐

## 六、TOFU/MOFU/BOFU 漏斗定位

| 阶段 | 判定标准 | 你的内容属于？ | 该阶段的正确策略 |
|------|---------|-------------|----------------|
| **TOFU 认知** | 目标是曝光，宽泛有趣高分享 | ☐ | CPM和品牌搜索量变化 |
| **MOFU 信任** | 目标是建立信任，深度干货案例 | ☐ | 互动率和私信获取量 |
| **BOFU 转化** | 目标是行动，限时优惠FAQ | ☐ | CPA和ROAS |

> ⚠️ 常见错误：明明是MOFU深度内容，却用TOFU的"博眼球"标题——导致点击高但信任流失。

## 七、自检终审

发布前最后5问：

1. **如果我是一个陌生人，在信息流中看到这篇，我会停下滑动吗？** ── 如果在"会"和"不会"之间犹豫，那就是不会。

2. **这篇内容的"不可替代性"在哪里？AI能不能在10秒内生成类似的内容？** ── 如果能，你的竞争优势是什么？

3. **这篇内容是在"寄生"别人的流量，还是在建立自己的流量资产？** ── 寄生型：蹭热度、追热点、纯信息搬运。资产型：强化个人标签、建立认知护城河。

4. **读完这篇的人，会不会主动分享给别人？** ── 如果答案不是100%的"会"，检查第二、三部分的缺失项。

5. **三个月后，这篇内容还有被阅读的价值吗？** ── 如果答案是否定的，你产出的可能只是"信息噪音"而非"知识资产"。

## 八、快速参考卡

### 爆款内容公式
`
爆款概率 = 认知冲突 × 情绪唤醒 × 分享动机 × 平台适配 ÷ 用户理解能耗
`

### 高唤醒情绪（优先使用）
敬畏 > 愤怒 > 焦虑 > 幽默

### 平台一句话口诀
- **小红书**：封面即标题，搜索即流量，真实即信任
- **抖音**：前3秒定生死，情绪化定传播，垂直人设定留存
- **公众号**：搜索权重定冷启，收藏价值定长尾，深度信任定转化
- **X/Twitter**：回复即流量，Thread即权重，中文即蓝海
