# 研报笔记

## 材料 1｜两会前瞻：人工智能+（2026-02-23）

### 0）材料类型识别

- 类型：主题投资型（叠加政策/行业框架）
- 依据：以两会政策定调与后续配套落地作为主驱动，目标是捕捉 2-3 个月政策催化行情；非单一公司盈利预测。

### 1）核心结论提取（三句话）

- 研究对象：两会“人工智能+”主线及其传导路径（算力基建 → 制造升级 → 能源优化）。
- 研究结论：政府工作报告定调后，相关板块大概率进入持续约 2-3 个月的主题行情；最直接受益为算力基础设施与国产替代链。
- 投资逻辑：政策强度/落地密度上升 → Capex 与订单预期上修 → 先交易预期差与估值抬升，后进入业绩验证与分化。

### 2）变量拆解（变量定义）

- 政策表述强度：政府工作报告中“人工智能+”排序、措辞、是否新增细分（如具身智能/一体化算力网）；决定主题高度。
- 政策落地密度：部委专项行动/试点名单/标准/资金工具的发布频率与速度；决定行情持续性与由普涨到分化的节奏。
- 算力资本开支（Capex）：运营商与互联网大厂在智算中心、GPU、服务器、光模块等投入规模与兑现节奏；决定硬件链景气。
- 国产替代渗透率：国产 AI 芯片/服务器/晶圆代工等从“可用”到“规模化”的进度；决定中期市占空间。
- 订单能见度：招投标、试点城市、采购比例、出货数据等；用于区分概念与真实受益。
- 盈利兑现能力：毛利率、产能利用率、产品结构升级与价格竞争；决定估值是否可持续。
- 估值位置：当前估值相对历史区间与增长兑现路径；决定预期差方向。

### 3）因果机制（至少两层因果链）

- 链条 A：两会定调“人工智能+”增强 → 专项行动/试点落地 → 智算中心建设与 Capex 上修 → GPU/服务器/光模块/IDC 需求上行 → 订单与收入兑现（滞后）+ 估值先行。
- 链条 B：“AI+制造”专项行动（场景/标杆企业/数据集）→ 智能工厂改造与机器人渗透 → 谐波减速器/功率半导体等核心部件国产化需求扩大 → 龙头份额提升/规模降本 → 利润改善或竞争压制下的分化。
- 链条 C：新型电力系统与容量电价机制完善 → 独立储能现金流属性增强 → 电网侧储能 ROI 改善/装机加速 → 系统集成商与设备订单上行 → 行业景气延长/估值切换。

### 4）资金结构分析（参与者与节奏）

- 提前布局：ETF 被动资金/部分机构通过半导体、算力 ETF 进行底部配置。
- 同步进入：游资/趋势/量化在 3/5 定调后追随推动“普涨-扩散”。
- 滞后跟随：稳健资金更偏好“政策确定 + 业绩拐点/订单验证”后向龙头集中。

### 5）价格与估值（是否已反映、预期差）

- 当前更偏“预期/估值驱动”，兑现存在时间滞后；核心预期差来自：政策超预期幅度、落地细则速度、国产化出货与良率数据。
- 主要风险：高估值透支未来 → 兑现不及预期时预期差转负。

### 6）催化剂（事件 + 周期）

- 短期（1-2 周）：3/5 政府工作报告；3/7 外交部长答问；3/8 部长通道（试点/标准/容量电价等细则）。
- 中期（1-3 月）：部委/地方配套政策密集出台、项目招标；一季报/业绩预告验证。
- 长期（6-12 月）：十五五任务推进（未来产业、关键核心技术攻关）。

### 7）风险与反证（逻辑失效条件）

- 政策不及预期或落地密度不足。
- 订单兑现慢/毛利率被价格战侵蚀。
- 技术迭代与出口限制导致国产替代节奏放缓。
- 扩产快于需求导致产能过剩与估值杀伤。

### 8）交易映射（成立/错误的资产）

- 若成立：算力基建链（芯片/服务器/光模块/IDC）、半导体国产替代链、AI+制造（机器人/自动化/工业软件）、储能（容量电价受益）。
- 若错误：高估值纯概念、缺订单验证的小市值题材回撤风险最大。

### 9）概率评估（区间 + 收益/风险比）

- 成立概率：60%–75%。
- 可能收益/风险比：约 1.2–2.0（偏主题交易，回撤同样可能较快）。

### 10）结构压缩（三句话）

- 核心变量：两会“人工智能+”表述强度与后续政策落地密度，决定 Capex/订单与国产替代节奏。
- 定价判断：短期以预期与估值交易为主，中期看业绩兑现；兑现慢则预期差转负。
- 交易标的：优先用龙头或 ETF 表达，回避高估值纯概念。

## 材料 2｜2026 年 3 月市场驱动全景：霍尔木兹封锁 + 两会 + 产业会议

### 0）材料类型识别

- 类型：宏观研究型（叠加主题投资型）
- 依据：以地缘事件为宏观冲击源，推演油价/运价/化工供给链，并叠加两会与产业会议形成“月度驱动地图”。

### 1）核心结论提取（三句话）

- 研究对象：霍尔木兹海峡封锁引发的能源与化工供给冲击，以及两会政策与产业会议对科技方向的催化。
- 研究结论：短期最硬主线是油运与油气（运距拉长 + 运价/油价弹性），化工品存在供给缺口交易；中期两会“能源安全+科技自主”提供接力。
- 投资逻辑：地缘事件先抬升风险溢价 → 价格快速上行（1-2 周）→ 若缓和则回吐；政策与会议对科技链更多偏情绪/预期驱动。

### 2）变量拆解（变量定义）

- 封锁持续时间：是否超过 1 周、是否升级为长期通道阻断；决定溢价维持。
- 外交/军事进展：任何缓和信号都可能触发油价与油运快速回撤。
- 全球需求强弱：需求走弱会抵消供给扰动对油价与化工品的中期支撑。
- 运距拉长与有效运力吸收：绕行导致有效运力下降；决定 VLCC 日租金弹性与持续性。
- 化工品供给缺口：甲醇/硫磺/溴素/尿素等出口中断对国内现货、库存与替代的影响。
- 两会能源安全与储备强度：决定上游增储上产、储备体系与仓储链条的中期政策 beta。
- 美联储与美元流动性：影响大宗与风险资产估值。

### 3）因果机制（至少两层因果链）

- 链条 A：封锁/冲突升级 → 供应中断预期 + 风险溢价上升 → 油价快速上冲 → 上游利润预期上修 → 油气股阶段性重估（缓和则回吐）。
- 链条 B：通道阻断 → 绕行运距拉长 → 有效运力被吸收、运价上行 → 油运公司利润弹性释放 → 资金短线集中、波动加剧。
- 链条 C：伊朗化工出口受阻 → 国内供给缺口/涨价 → 煤化工/一体化低成本企业利润改善 → 化工板块补涨（受需求与调控影响）。

### 4）资金结构分析

- 提前布局：事件驱动资金抢跑“油运/油气/紧缺化工”。
- 同步进入：趋势与量化在价格突破后跟随放大波动。
- 滞后跟随：稳健资金更可能在两会确认能源安全/储备后配置上游与仓储等中期逻辑。

### 5）价格与估值（是否已反映、预期差）

- 油价/油运属于风险溢价交易：上行快、回吐也快。
- 最大预期差：封锁持续时间与升级程度；若出现可信缓和路径，预期差迅速转负。

### 6）催化剂（事件 + 周期）

- 短期（1-2 周）：外交与军事动态；OPEC/储备释放表态；运价与油价高频数据；两会期间能源安全新增表述。
- 中期（1-3 月）：两会后配套项目落地（储备体系、上游投资）；化工品进口替代与开工率变化；美联储议息对风险偏好影响。

### 7）风险与反证（逻辑失效条件）

- 封锁快速解除或被增产/替代通道/释放储备对冲。
- 全球需求明显走弱。
- 行政调控抑制价格与利润传导。
- 拥挤交易导致回撤放大。

### 8）交易映射

- 若成立：油运（弹性最直接）、油气开采、特定紧缺化工替代、仓储与储备。
- 若错误：油运与短期涨幅过大的事件驱动品种回撤风险最大。

### 9）概率评估（区间 + 收益/风险比）

- 地缘冲击主线成立概率：45%–65%；收益/风险比约 0.8–2.5（高度依赖入场时点与持续时间）。
- 政策接力主线成立概率：60%–75%；收益/风险比约 1.0–2.0。

### 10）结构压缩（三句话）

- 核心变量：封锁持续/升级决定油价与油运溢价高度；两会能源安全与储备加码决定中期接力。
- 定价判断：事件溢价交易易反转；更稳的是政策驱动的储备与上游投资链。
- 交易标的：短线油运/上游，中期能源安全（上游+储备）与科技自主可组合表达，严控拥挤与流动性风险。
