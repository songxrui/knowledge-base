# 一、五条总纲

1. **先停损，再优化。** 先砍熬夜 / 短视频 / 奶茶 / 油炸 / 冲动消费，再谈补剂、护肤、训练进阶。
2. **系统胜过意志。** 让环境、默认规则、SOP 替代临场硬扛。
3. **一次只动一个变量。** 多变量同改 = 无法归因。
4. **80% 高分食物 / 动作 + 20% 自由额度。** 没有自由额度，系统会反弹。
5. **能否低摩擦持续 1 年？** 否，则砍掉。

# 二、每日六件事（MED 打卡）

这 6 件事覆盖 80% 的健康收益。其他都是锦上添花。

- [ ]  **固定起床时间**（误差 ≤30 分钟，含周末）
- [ ]  **醒后 10 分钟户外光**（阴天也算，隔窗户不算）
- [ ]  **完成一次运动**（力量 / Zone 2 / 步行任选其一）
- [ ]  **蛋白 ≥130 g + 三色蔬菜**
- [ ]  **晚间牙线 / 冲牙器 1 次**
- [ ]  **白天出门防晒**

# 三、三段式协议（晨 / 日 / 夜）

## 🌅 起床 30 分钟（开机程序）

- 不碰手机 30–60 分钟
- 500 ml 温水
- 户外光 5–10 分钟
- 5 分钟颈胸活动 + 下颌放松

## ☀️ 日间（运行）

- 上午 9–12 点用皮质醇高峰做高认知任务，不刷手机
- 咖啡因截止 **下午 2 点**
- 久坐每 50 分钟起身 2 分钟（脚本：站起 + 鼻呼吸×3 + chin tuck×10s + 肩胛后收×10 + TMJ 巡检）
- 午餐先菜→蛋白→碳水 + 醋淋菜
- 餐后散步 10 分钟
- 下午 1–2 点小睡 ≤20 分钟（必要时）

## 🌙 睡前 1 小时（关机程序）

- 物理隔离蓝光（手机离床 / 暖光 / 夜间模式）
- 睡前 3 小时不进食大餐
- 睡前 2 小时不剧烈运动
- 卧室 18–20°C、完全遮光
- 把焦虑写进 Inbox，清空大脑

# 四、训练（每周）

## 力量（每周 3 次：周一 / 三 / 五）

以《训练 6.0》为唯一权威动作模板。本页只放执行规则。

- **上下牙不接触**
- 复合动作 RIR 2–3，单关节 RIR 0–1
- 组间休息：复合 2–3 分钟，孤立 60–90 秒
- 同动作连续 2 次达上沿且不变形 → 上肢 +1 kg，下肢 +2.5–5 kg
- 不冲到完全力竭。"开始咬牙硬扛" = 下次该动作减重 5–10%

## 有氧

- **Zone 2（鼻呼吸、可说话）**：≥150 分钟/周（快走 / 骑行 / 慢跑）
- **HIIT / Zone 5**：1–2 次/周，每次 20–30 分钟
- 步数：≥8000 步/天（全因死亡率拐点 7000–8000）

## 训练日红线

任一条触发 → 自动降级，不讨论：

- TMJ 7 天均值 ≥5/10 → 下肢减重 10–20%，RIR 锁 3，取消所有冲失败倾向，只保留侧平举 / 下拉 / 划船 / face pull / 核心，持续 2 周
- 关节痛 ≥3/10 持续 3 天 → 该动作减重 10–20% 或替换
- 锁颌 / 张口 ≤3 指 → 立即就医评估

## 3｜两餐模板（中午起床）

| 餐次 | 组合 | 目标 |
| --- | --- | --- |
| 第一餐 | 燕麦 50 g + 酵母蛋白 30 g + 脱脂牛奶 400 ml + 香蕉 1 + 鸡蛋 2 + 蔬菜 300–500 g + 坚果 10 g/橄榄油 5–10 g | 700–850 kcal；蛋白 55–65 g；负责供能和抗暴食 |
| 第二餐 | 酵母蛋白 30 g + 黄黑豆粉 30–50 g + 苹果 1 + 鸡蛋 1 或鸡胸/鱼/虾/瘦牛肉 150 g + 蔬菜 500 g + 燕麦 0–30 g | 650–850 kcal；蛋白 60–75 g；训练日加燕麦，低活动日少加/不加 |

## 4｜瘦脸控制点

- **低钠优先级 &gt; 继续饿**：茶叶蛋 ≤1；少外卖、卤味、酱油、辣酱、咸菜、袋装零食。
- **高钾+水**：绿叶菜 500 g+，番茄/黄瓜/菌菇，香蕉或土豆/红薯，水 2–2.5 L。
- **不练咬肌**：不嚼口香糖、不啃硬物、不咬牙；训练保持“唇闭、齿离、舌上、鼻吸”。
- **作息稳定**：起床误差 ≤60 分钟；脸肿先查钠、睡眠、水、碳水波动、咬牙。

## 5｜动态调整

| 信号 | 动作 | 禁止 |
| --- | --- | --- |
| 连续 14 天 7 日均体重不降 | 每日 -100 kcal 或步数 +2000；优先减燕麦/米饭/红薯 | 不减蛋白，不把脂肪压到 40 g 以下 |
| 一周下降 &gt;0.9 kg 且训练变差 | 每日 +150–250 kcal；优先加燕麦/米饭/红薯 | 不硬扛，不加奶茶零食 |
| 第 7 周后冲刺/训练下滑 | 每周 1 天高碳日：额外 +80–120 g 碳水，只用燕麦/米饭/红薯 | 不用火锅、甜品、炸物当高碳日 |
| 脸肿但体重下降 | 查钠、睡眠、水、碳水忽高忽低、咬牙/TMJ | 不要继续盲目降热量 |

## 6｜每周验收

- **记录**：7 日均体重、腰围、正脸照、侧脸照、训练表现、饥饿/暴食冲动、睡眠。
- **有效**：每周 -0.4–0.7 kg + 腰围下降 + 脸水肿减少 + 训练不崩。
- **停止线**：已到 10–12% 且脸明显瘦时优先维持；长期压 9% 对睡眠、情绪、性激素、训练恢复压力大。

## 补剂三段

- **早**：多维 + 鱼油（EPA 688 + DHA 344）+ 维生素 D3 + B12 + 菊粉 5–8 g
- **训前**：乳清 30 g + 肌酸 5 g
- **晚**：甘氨酸镁 + 锌 + Q10 + 花青素 + 虾青素 12 mg + 姜黄 + 洋车前子壳 5 g

# 六、肠道 / 痤疮 / 肾保护（专项）

- **纤维渐进**：菊粉从 5 g 起，每周加 2 g，上限 10 g；与洋车前子壳间隔 ≥1 h
- **触发物管理**：西兰花 → 用花菜 / 包菜替代；高果糖（葡萄干）少碰
- **久坐中断**：每 30–45 分钟起身 → 减肛周静脉压力
- **痤疮**：白米饭部分换红薯 / 紫薯；4 周减牛奶保留酸奶观察；锌 + Omega-3 + 防晒
- **肾保护**：水 ≥2 L、柠檬水每日、控钠 <2300 mg、不乱吃止痛药 / 不明中药、每年验尿 + 胱抑素 C

# 七、口腔（每日）

- 含氟牙膏×2 次×2 分钟（巴氏刷牙法）；刷后**少漱口**
- **牙线 / 冲牙器 1 次/天**（比刷牙更重要——牙刷只清洁 60% 表面）
- 不在饭后 30 分钟内刷牙（酸性环境损伤釉质）
- 训练时**唇闭、齿离、舌上、鼻吸**
- 每 6 个月洁牙 + 检查
- 红旗：刷牙出血 >2 周 / 牙痛冷热痛 / 张口弹响痛 → 看牙

# 八、皮肤（每日）

- **早**：温和洁面 + 保湿 + **SPF 30+ 防晒**（出门就涂，阴天也涂）
- **晚**：温和洁面 + 保湿（含神经酰胺）
- 8 周内只稳定 1 个核心成分（维 A / BPO / 壬二酸三选一）
- 顺序：**先控新痘 → 再淡痘印 → 再改善暗沉**
- 单选时只做一件 → **防晒**（光老化占 80%+）

# 九、抗成瘾三硬规则

1. **手机不进床。** 离床充电。
2. **非刚需购物 48 小时清单。** 想买的写进 48 小时观察池，过期才买。
3. **奶茶每周最多 1 次**，小杯、不加小料。

冲动急救脚本（30 秒）：

停 10 分钟 → 命名冲动 → 定位（饿 / 累 / 烦 / 孤独 / 压力 / 无聊）→ 走 3 分钟 / 喝水 / 洗脸 / 写触发 → 记录结果。

# 十、急救手册（突发不适）

## 红绿灯判断

| 等级 | 特征 | 行动 |
| --- | --- | --- |
| 绿色 | 不适可控、症状轻、功能未受限 | 按下方 SOP + 观察 24–48h |
| 黄色 | 持续恶化 / 影响功能 / 反复发作 | 先自救 → 48h 未改善 → 就医 |
| 红色 | 剧烈疼痛 / 呼吸困难 / 意识改变 / 大量出血 / 胸痛 | **立即 120 或急诊** |

## 常见处理

- **感冒**：水 2.5–3 L + 维 C 2 g + 锌 30 mg（24 h 内最有效）+ 蜂蜜柠檬水 + 盐水洗鼻；高热 >38.5°C 用对乙酰氨基酚 / 布洛芬
- **头痛**：先排红色信号（一生中最严重 / 颈强 + 发热 / 突发视觉变化 → 立即急诊）；紧张型 → 500 ml 水 + 颈拉伸 + 太阳穴薄荷油
- **过度换气**：生理性叹息×3–5；缓慢腹式呼吸（吸 4 / 屏 4 / 呼 6）
- **腹泻**：补水 + 口服补液盐（1 L 水 + 6 茶匙糖 + ½ 茶匙盐）；BRAT 饮食；不轻易用洛哌丁胺；血便 / 高热 → 就医
- **扭伤**：用 PEACE & LOVE 取代 RICE。**前 48 h 不用冰敷不用布洛芬**（炎症是修复起点）
- **割伤**：流动清水冲 5 分钟 → 碘伏（不用酒精 / 双氧水）→ 无菌敷料；伤口 >2 cm / 关节处 / 动物咬伤 → 就医
- **心悸**：胸痛 / 晕厥 → 立即急诊；良性心悸 → Valsalva 动作（深吸闭嘴捏鼻用力 15 s）+ 冰水敷脸 + 缓慢深呼吸

## 家庭急救箱（最小集合）

对乙酰氨基酚 / 布洛芬 / 口服补液盐 / 蒙脱石散 / 氯雷他定 / 碘伏 / 创可贴 + 无菌纱布 / 弹性绷带 / 电子体温计。

# 十一、年度筛查清单（21 岁起）

一次性硬投入，终身收益。

- [ ]  确认乙肝疫苗已接种（表面抗体阳性）
- [ ]  幽门螺杆菌呼气试验（阳性→根除）
- [ ]  HPV 疫苗（男性也推荐）
- [ ]  **基线血液检查**：血常规 + 肝肾功 + 血脂 + **ApoB** + **Lp(a)（一生测一次即可）**+ 空腹胰岛素 + HbA1c + hs-CRP + 维生素 D + 同型半胱氨酸
- [ ]  激素基线：晨起睾酮 + SHBG + DHEA-S
- [ ]  身体组成：DEXA 扫描（一次基线）
- [ ]  口腔检查 + 洁牙
- [ ]  皮肤痣自检（ABCDE 规则）

## 最优指标参考

- ApoB <90 mg/dL（理想 <60）
- 血压 <120/80
- 空腹血糖 <5.0 mmol/L、HbA1c <5.4%
- hs-CRP <1.0 mg/L
- 空腹胰岛素 <5 μIU/mL
- 静息心率 50–65 bpm
- 维生素 D 血清 40–60 ng/mL

# 十四、降级协议（红线触发，自动执行不讨论）

出现以下任一情况，按对应方案降级 7–14 天，不与自己讨论意志力：

- **睡眠 <6 h 连续 3 天** → 砍训练量 30–40%、取消 HIIT、咖啡因截止提前到中午
- **TMJ ≥5/10** → 训练降级方案（见第四节）
- **持续动力缺失** → 只做最小动作：出门走 10–20 分钟 + 远离高刺激 + 1 个 25 分钟最小任务
- **频繁感冒 / hs-CRP 持续 >1** → 暂停高强度，专注睡眠 + Omega-3 + 减 UPF
- **HRV 持续走低 + 静息心率走高** → 增加休息日，砍训练量
