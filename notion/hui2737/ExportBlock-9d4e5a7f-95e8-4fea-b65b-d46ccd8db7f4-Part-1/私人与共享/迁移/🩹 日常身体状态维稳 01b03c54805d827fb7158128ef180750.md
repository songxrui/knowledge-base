# 一、日常身体退化的隐形驱动因子

<aside>
🔍

**你感受到的「身体在变差」通常不是某个器官出了问题，而是几个底层系统在慢性失调。** 找到并修复这些失调，大多数不适会自动消失。

</aside>

## 1.1 七大日常退化驱动因子

| **驱动因子** | **导致的日常不适** | **你可能没意识到的信号** |
| --- | --- | --- |
| **慢性脱水** | 头痛、疲劳、注意力下降、便秘、皮肤干燥、关节僵硬 | 尿液深黄色、嘴唇干裂、下午无故疲惫 |
| **微量营养素缺乏** | 免疫力低下、反复感冒、伤口愈合慢、脱发、情绪波动 | 指甲脆弱/有白点、口角炎、频繁口腔溃疡 |
| **睡眠债累积** | 认知模糊、反应变慢、情绪失控、免疫下降、体重增加 | 需要闹钟才能醒、醒后 30 min 仍困、周末补觉 >2h |
| **姿势性损伤累积** | 颈椎痛、腰痛、头痛、圆肩驼背、呼吸浅 | 久坐后站起来需要「缓一下」、无法舒适地深呼吸 |
| **肠道功能紊乱** | 腹胀、便秘/腹泻交替、皮肤问题、情绪低落、食物不耐受 | 排便不规律、放屁恶臭、饭后腹胀明显 |
| **自主神经失衡（交感过度激活）** | 焦虑、心悸、失眠、消化不良、肌肉紧张、易怒 | 静息心率偏高（>75 bpm）、手脚冰凉、磨牙 |
| **运动不足 / 运动过度** | 僵硬、疲劳、免疫下降、慢性疼痛、恢复变慢 | 爬楼气喘、拎重物吃力、运动后 48h+ 仍酸痛 |

# 二、日常防退化协议：分系统 SOP

## 2.1 💧 水合系统

<aside>
💧

**慢性轻度脱水是最被低估的日常不适来源。** 仅 2% 的脱水即可导致认知能力下降 20%、体能下降 25%。

</aside>

### 执行规则

- **每日饮水量**：63kg × 30-35 mL =2205
- **起床后第一件事**：250-500 mL 温水（补充 8h 睡眠蒸发量）
- **运动前中后**：运动前 30 min 喝 300 mL → 运动中每 15 min 小口补 150 mL → 运动后按失重量 1.5 倍补充
- **咖啡因/酒精修正**：每杯咖啡额外补 150 mL 水；每单位酒精额外补 250 mL 水
- **电解质**：如果大量出汗，加入钠（盐）、钾、镁（运动饮料或自制：1L 水 + 1/4 tsp 盐 + 少量柠檬汁）
- **判断指标**：尿液颜色 = 淡黄色为最佳；深黄 = 脱水；完全透明 = 可能过量

## 2.2 🦴 肌骨系统（抗久坐 / 抗姿势退化）

<aside>
🦴

**久坐是 21 世纪的吸烟。** 每天 8h+ 久坐者全因死亡率增加 60%。但关键不是「坐多久」，而是「多久动一次」。

</aside>

### 日常防退化 SOP

**每 45-60 分钟起身活动 2-3 分钟（设定时器）：**

- **站立 + 髋部绕环 × 5**
- **胸椎旋转拉伸 × 5 每侧**
- **靠墙天使 × 10（纠正圆肩）**
- **深蹲到底部停留 30 秒（髋关节灵活性）**

**每日必做（5-10 min）**：

- [ ]  **颈椎**：颈部 4 方向拉伸（前/后/左/右，各 30 秒）+ 收下巴练习 × 15
- [ ]  **胸椎**：泡沫轴胸椎放松 2 min + 猫牛式 × 10
- [ ]  **腰椎/髋**：90/90 髋关节拉伸 + 鸽子式各 60 秒 + 麦吉尔三大（curl-up、side plank、bird dog）
- [ ]  **腕部**（键盘使用者）：手腕绕环 + 祈祷式拉伸 + 反向祈祷式各 30 秒

**工位设置**：

- 屏幕上缘与眼睛平齐
- 前臂与桌面平行
- 双脚平放地面
- **考虑升降桌：坐站交替（50 min 坐 / 10 min 站）**

### 常见疼痛预防

| **部位** | **常见原因** | **预防策略** |
| --- | --- | --- |
| **颈椎** | 头前伸姿势（看手机/屏幕） | 收下巴练习 + 屏幕高度调整 + 每小时活动 |
| **下背/腰** | 久坐 + 核心弱 + 髋屈肌紧 | 麦吉尔三大核心训练 + 髋屈肌拉伸 + 臀桥激活 |
| **肩膀** | 圆肩 + 胸肌紧 + 上背弱 | 面拉（face pull）+ 胸肌拉伸 + 肩外旋训练 |
| **手腕** | 长时间键盘/鼠标使用 | 人体工学鼠标 + 腕部拉伸 + 前臂强化 |
| **膝盖** | 股四头肌/臀肌弱、下肢力线不正 | 深蹲模式纠正 + 单腿训练 + 不跳过腿部训练日 |

## 2.3 🫁 呼吸系统

> **多数人都在错误地呼吸。** 口呼吸、浅呼吸、胸式呼吸 → 交感神经过度激活 → 焦虑/疲劳/睡眠差的恶性循环。
> 

### 呼吸优化规则

1. **鼻呼吸为默认**（日间和睡眠都应该鼻呼吸）
    - 鼻腔过滤空气 + 加温加湿 + 产生一氧化氮（扩血管）
    - 如果睡觉张嘴 → 考虑口呼吸贴
2. **腹式呼吸为基础**
    - 吸气时腹部向外膨胀（横膈膜下降）
    - 呼气时腹部自然回收
    - 每天练习 3-5 min：手放腹部，感受起伏
3. **呼吸频率降到 5-6 次/分钟**（正常人 12-20 次）
    - 刻意慢呼吸 → 迷走神经激活 → 副交感主导 → 放松
    - 方法：吸 4 秒 → 呼 6 秒 → 循环 5 min
4. **高压时刻使用生理性叹息：双吸一长呼（吸-吸-呼），1-3 次即可降低应激反应**

## 2.4 🍽️ 消化系统日常维护

### 减少消化不适的规则

1. **吃饭速度**：每口咀嚼 15-20 次 → 消化始于口腔（唾液淀粉酶）
2. **进食环境**：坐着吃、不焦虑地吃 → 副交感神经主导时消化效率最高
3. **餐间距离**：至少 3-4 小时 → 给消化间期清扫波（MMC）运作的时间 → 清理肠道残渣
4. **腹胀应对**：
    - 饭后散步 10-15 min（最有效）
    - 减少产气食物（豆类、十字花科蔬菜需逐步增量）
    - 薄荷茶/姜茶辅助
5. **便秘预防**：
    - 纤维 25-35g/天（逐步增加，不要突然大量）
    - 充足饮水
    - 晨起温水 + 咖啡（刺激结肠蠕动）
    - 如厕姿势：脚踩矮凳模拟蹲姿（35° 角打开耻骨直肠肌）
6. **食物不耐受排查**：如果反复出现特定不适 → 2 周排除饮食法（去除可疑食物 → 逐一回添 → 观察反应）

不耐受：西兰花，高果糖（葡萄干）

## 2.5 🛡️ 免疫系统日常维护

### 减少生病频率的策略

| **层级** | **策略** | **机制** |
| --- | --- | --- |
| **阻断传播** | 勤洗手（20 秒 + 肥皂）；不摸脸；人群密集处戴口罩 | **80% 感染通过手→面部传播** |
| **粘膜防线** | **鼻腔冲洗（盐水洗鼻）**；保持室内湿度 40-60% | 干燥粘膜更易被病原穿透 |
| **充足睡眠** | 7-9h → NK 细胞活性正常 | 一晚 4h 睡眠 → NK 活性降 70% |
| **适度运动** | 30-60 min 中等强度 → 免疫细胞巡逻增加 | 运动后免疫监视窗口增强数小时 |
| **不过度训练** | 避免连续高强度训练无恢复 → 免疫抑制窗口 | 极高强度运动后 3-72h 免疫功能暂时下降 |
| **维生素 D** | 2000-4000 IU/天或充足日照 | 免疫系统的总开关之一 |
| **锌** | 15-30 mg/天（食物或补充） | 免疫细胞增殖和功能必需 |
| **维生素 C** | 500-1000 mg/天（感冒时可增至 2g） | 中性粒细胞功能支持 |
| **肠道健康** | 70% 免疫系统在肠道 → 纤维 + 发酵食品 | 肠道菌群训练免疫系统 |
| **压力管理** | 慢性压力 → 皮质醇 → 免疫抑制 | 压力下感冒概率翻倍 |

### 换季/感冒高发期强化

- 出门戴口罩（尤其公共交通）
- 回家立即洗手 + 换衣服
- 每日盐水洗鼻
- 维生素 C + 锌剂量适当提升
- 保证睡眠 → 这是最强免疫武器

## 2.6 🧠 精力与认知维稳

### 日间精力崩塌的常见原因与对策

| **症状** | **最可能原因** | **即时解法** | **根本解法** |
| --- | --- | --- | --- |
| 下午 2-3 点昏沉 | 午餐血糖飙升后崩塌 | 饭后散步 10 min + 冷水洗脸 | 午餐减少精制碳水 + 增加蛋白质/纤维 |
| 整天都困倦 | 睡眠质量差 / 睡眠呼吸暂停 | 20 min 午休（不超过 30 min） | 改善睡眠卫生 / 排查呼吸暂停 |
| 注意力涣散 | 多巴胺基线低 / 信息过载 | 5 min 户外散步 + 阳光 | 减少短视频 + 90 min 专注周期训练 |
| 情绪波动大 | 血糖不稳 / 睡眠不足 / 压力累积 | 生理性叹息 + 吃一小份坚果 | 稳定血糖饮食 + 睡眠优先 + 压力释放 |
| 无故焦虑/心慌 | 咖啡因过量 / 交感过度激活 | Box breathing（4-4-4-4 秒）× 5 轮 | 减少咖啡因 + 每日迷走神经训练 |
| 频繁头痛 | 脱水 / 颈椎问题 / 眼睛疲劳 / 睡眠不足 | 喝 500 mL 水 + 颈部拉伸 | 纠正水合习惯 + 每 20 min 远眺 20 秒 |

### 每日精力管理框架

1. **起床后 30 min**：晨光暴露 10 min + 250 mL 水 + 轻度活动 → 皮质醇自然唤醒峰
2. **上午 9-12 点**：利用皮质醇高峰做高认知负荷工作 → 不刷手机
3. **午餐**：高蛋白 + 中等脂肪 + 低精制碳水 + 蔬菜 → 避免血糖过山车
4. **午后 1-2 点**：20 min 午休（设闹钟防睡过头） 或 饭后散步
5. **下午 3-5 点**：第二高效窗口 → 咖啡因截止时间不晚于下午 2 点
6. **傍晚**：运动的最佳窗口（体温和协调性最高）
7. **晚上 9-10 点起**：减少蓝光 + 降低环境光 → 褪黑素启动

## 2.7 👀 眼睛与视力维护

- **20-20-20 规则**：每 20 分钟 → 看 20 英尺（6 米）远 → 持续 20 秒
- **户外时间**：每天 1-2h → **自然光是预防近视加深的最强干预**（多巴胺抑制眼轴增长）
- **屏幕距离**：至少 50 cm
- **夜间模式**：晚上 **8 点后开启暖色滤光**
- **干眼缓解**：刻意眨眼（看屏幕时眨眼频率降低 60%）+ 热敷睑板腺 + Omega-3

## 2.8 🦷 口腔维护

> ***口腔健康直接影响全身健康*。** 牙周病与心血管疾病、糖尿病、认知退化直接相关。
> 
- **刷牙**：每天 2 次 × 2 min，45° 角巴氏刷牙法，软毛牙刷
- **牙线**：***每天 1 次（比刷牙更重要的是牙线！邻面龋靠刷牙无法预防）***
- **漱口水**：非必须，仅在牙周病治疗期间使用含氯己定的款
- **洗牙**：每年 1-2 次专业洁治
- **不做的事**：不用牙齿开瓶盖/咬硬物、不在饭后 30 min 内刷牙（酸性环境下刷牙损伤牙釉质）

## 2.9 🧴 皮肤维护

### 最简有效护肤 SOP

1. **清洁**：每天 1-2 次温和洁面（避免皂基）→ 温水而非热水
2. **保湿**：洗完脸后 30 秒内涂保湿霜 → 锁住水分
3. **防晒**：SPF 30+ 广谱防晒霜（每天，不只是晴天）→ 紫外线是 80% 皮肤老化的原因
4. **不要过度清洁**：破坏皮肤屏障 → 反而导致出油/敏感
5. **如果*只能做一件事 → 防晒***

# 三、突发不适急救手册

<aside>
🚨

**以下是常见突发生理不适的分级应对方案。** 原则：先判断严重程度 → 能自救的自救 → 需要就医的立即就医 → 不拖。

</aside>

## 3.1 判断框架：自救 vs 就医

| **等级** | **特征** | **行动** |
| --- | --- | --- |
| **绿色 — 自救** | 不适可控、症状轻微、功能未受限 | 按下方对应 SOP 处理 + 观察 24-48h |
| **黄色 — 观察+可能就医** | 症状持续恶化、影响日常功能、反复发作 | 先自救缓解 → 48h 未改善 → 就医 |
| **红色 — 立即就医/急救** | 剧烈疼痛、呼吸困难、意识改变、大量出血、胸痛 | **立即 120 或去急诊，不要等** |

## 3.2 常见突发不适 SOP

### 🤒 感冒 / 上呼吸道感染

- **感冒应对方案**（自限性疾病，7-10 天自愈）
    
    **立即行动**：
    
    - **大量饮水（每天 2.5-3L 温水）**
    - 休息 → 这不是偷懒，这是免疫系统在战斗
    - 维生素 C 2000 mg/天 + 锌 30 mg/天（发病 24h 内开始最有效）
    - 蜂蜜 + 柠檬 + 温水（缓解咽痛，蜂蜜止咳效果 ≥ 止咳糖浆）
    - 盐水漱口（缓解咽痛）+ 盐水洗鼻（缓解鼻塞）
    
    **对症处理**：
    
    - 发热 > 38.5°C → 对乙酰氨基酚（泰诺）或布洛芬
    - 鼻塞严重 → 0.05% 羟甲唑啉滴鼻（不超过 3 天！否则反弹性鼻塞）
    - 咳嗽 → 蜂蜜 > 大多数 OTC 止咳药
    
    **⚠️ 就医信号**：
    
    - 高热 > 39°C 持续 3 天+
    - 呼吸困难
    - 黄绿色浓痰（可能细菌感染）
    - 单侧面部疼痛（可能鼻窦炎）

### 🤕 头痛

- **头痛分型与应对**
    
    **第一步：排除红色信号**
    
    - 「一生中最严重的头痛」→ **立即急诊**（可能蛛网膜下腔出血）
    - 伴发热 + 颈部僵硬 → **立即急诊**（可能脑膜炎）
    - 突发视力改变 + 头痛 → **尽快就医**
    
    **常见类型与自救**：
    
    **紧张型头痛**（最常见，双侧压迫感）：
    
    - 喝 500 mL 水（排除脱水）
    - 颈部拉伸 + 太阳穴按摩
    - 薄荷油涂太阳穴
    - 如需止痛：对乙酰氨基酚 500 mg 或布洛芬 400 mg
    
    **偏头痛**（单侧搏动性，可能有先兆）：
    
    - 立即转入安静暗室
    - 冰敷颈后或额头
    - 尽早服用止痛药（偏头痛越早吃药越有效）
    - 咖啡因有时有效（小杯浓缩咖啡）
    - 频繁发作 → 就医评估预防性治疗
    
    **根本预防**：规律睡眠 + 充足饮水 + 减少屏幕时间 + 颈椎维护
    

### 🫁 呼吸急促 / 过度换气

- **呼吸不适应对**
    
    **排除严重问题**：
    
    - 胸痛 + 呼吸困难 → **立即急诊**（可能心脏问题/气胸/肺栓塞）
    - 哮喘发作 → 使用急救吸入器 → 无缓解立即急诊
    
    **焦虑/恐慌导致的过度换气**（最常见）：
    
    1. 停下你正在做的事
    2. **生理性叹息**：鼻吸-鼻吸-口呼（长呼气），重复 3-5 次
    3. 缓慢腹式呼吸：吸 4 秒 → 屏 4 秒 → 呼 6 秒
    4. 如果头晕手麻（CO₂ 过低）→ 对着双手合拢形成的空间呼吸（重新吸入 CO₂）
    5. 扎根练习：数周围 5 个看到的东西、4 个听到的声音、3 个触碰到的物体

### 🤢 恶心 / 呕吐

- **恶心呕吐应对**
    
    **排除严重问题**：
    
    - 呕吐物含血（咖啡渣样或鲜红色）→ **立即急诊**
    - 剧烈腹痛 + 呕吐 → 可能急腹症 → **就医**
    - 头部受伤后呕吐 → **立即急诊**
    
    **常见自救**：
    
    - 暂停进食 2-4h → 让胃休息
    - 小口缓慢补液（电解质水 > 白水）→ 防脱水
    - 生姜（姜茶/姜片/姜糖）→ 循证止吐效果
    - 薄荷茶
    - 恢复进食时从 BRAT 饮食开始：香蕉、米饭、苹果泥、吐司
    - 避免油腻/辛辣/乳制品直到完全恢复
    
    **晕车/晕船**：
    
    - 看远处固定点（地平线）
    - 生姜提前 30 min 服用
    - 如需药物：苯海拉明（晕海宁）提前 30 min 服
    - 按压内关穴（手腕内侧三横指处）

### 💩 急性腹泻

- **腹泻应对方案**
    
    **排除严重问题**：
    
    - 血便 → **就医**
    - 高热 + 腹泻 → **就医**
    - 严重脱水症状（少尿、头晕、口干严重）→ **就医**
    
    **自救策略**：
    
    - **首要任务 = 补水补电解质**（腹泻的危险不在于拉肚子本身，而在于脱水）
    - ***口服补液盐（ORS）：1L 水 + 6 tsp 糖 + 1/2 tsp 盐（WHO 配方）***
    - ***少量多次饮用（每 5-10 min 一小口）***
    - ***暂时避免乳制品、高纤维、咖啡因***
    - BRAT 饮食过渡
    - **益生菌可缩短病程 1 天左右**
    - 一般 2-3 天自愈；超过 3 天不改善 → 就医
    
    **不建议立即使用止泻药**（洛哌丁胺）——腹泻是身体排出病原的防御反应。只有在必须出门/无法忍受时才用。
    

### 🔥 发热

- **发热处理方案**
    
    > **发热是免疫反应，不是敌人。** 37.5-38.5°C 的低热不必急于退烧——此时免疫系统效率最高。
    > 
    
    **分级处理**：
    
    | 温度 | 行动 |
    | --- | --- |
    | 37.5-38.5°C | 多饮水 + 休息 + 监测 → 通常不需退烧药 |
    | 38.5-39.5°C | 物理降温（温水擦浴、退热贴）+ 对乙酰氨基酚 500 mg 或布洛芬 400 mg |
    | > 39.5°C 或持续 3 天+ | **就医** |
    | > 40°C 或伴意识改变 | **立即急诊** |
    
    **注意**：
    
    - 对乙酰氨基酚和布洛芬**可以交替使用**（间隔 4h）
    - 对乙酰氨基酚每日不超过 3g（伤肝）
    - 布洛芬需饭后服用（伤胃）
    - **不要用酒精擦浴降温**（可能经皮肤吸收）
    - 多饮水 → 发热时水分蒸发量大增

### 💪 肌肉拉伤 / 扭伤

- **PEACE & LOVE 方案**（已取代旧的 RICE 方案）
    
    **急性期（0-72h）→ PEACE**：
    
    - **P**rotect（保护）：停止导致疼痛的活动
    - **E**levate（抬高）：受伤部位高于心脏
    - **A**void anti-inflammatories（避免抗炎药）：前 48h 不用布洛芬/冰敷 → 炎症是修复的起点！
    - **C**ompress（加压）：弹性绷带适度加压
    - **E**ducate（教育自己）：大多数损伤不需要影像检查，身体会自愈
    
    **恢复期（72h 后）→ LOVE**：
    
    - **L**oad（适度负荷）：在不痛的范围内逐渐恢复活动
    - **O**ptimism（乐观心态）：积极心态加速恢复（有证据支持）
    - **V**ascularisation（促进血液循环）：轻度有氧运动促进血流
    - **E**xercise（锻炼）：逐步恢复训练，循序渐进
    
    **⚠️ 就医信号**：
    
    - 完全无法负重 → 可能骨折
    - 关节明显畸形 → 可能脱位
    - 严重肿胀不消退 → 可能韧带撕裂

### 🔪 割伤 / 擦伤

- **伤口处理 SOP**
    1. **止血**：干净纱布/布料直接按压 10-15 min → 不要反复掀开看
    2. **清洁**：流动清水冲洗 5 min → 去除污物
    3. **消毒**：碘伏（不是酒精或双氧水！它们杀伤力过大，延缓愈合）
    4. **覆盖**：无菌敷料或创可贴 → 湿润愈合比暴露愈合快 30-50%
    5. **更换**：每天换敷料，观察感染迹象
    
    **⚠️ 需要就医缝合的情况**：
    
    - 伤口 > 2cm 或深及脂肪层
    - 伤口边缘无法自然对合
    - 面部伤口（美观考虑）
    - 关节处伤口
    - 动物咬伤
    - 铁锈/脏物污染的深伤口 → 确认破伤风疫苗是否有效

### 🫀 心悸 / 心跳异常

- **心悸应对**
    
    **排除紧急情况**：
    
    - 胸痛 + 心悸 → **立即急诊**
    - 晕厥/接近晕厥 → **立即急诊**
    - 持续快速不规则心跳 > 15 min → **就医**
    
    **常见良性心悸的自救**：
    
    - **Valsalva 动作**：深吸气后闭嘴捏鼻用力呼气 15 秒 → 刺激迷走神经 → 减慢心率
    - **冷水激活**：冰水敷脸 30 秒 或 将脸浸入冰水 → 潜水反射 → 心率降低
    - **缓慢深呼吸**：吸 4 秒 → 呼 8 秒 × 10 轮
    - **减少诱因**：咖啡因、酒精、睡眠不足、脱水、压力

### ⚡ 抽筋

- **肌肉痉挛处理**
    
    **即时处理**：
    
    - 轻柔拉伸痉挛肌肉（小腿抽筋 → 脚趾向胫骨方向扳）
    - 局部按摩
    - 热敷（缓解痉挛后）
    
    **常见原因与预防**：
    
    - 脱水 → 充足饮水
    - 电解质失衡（钠/钾/镁不足）→ 运动后补充电解质
    - 运动前热身不足 → 5 min 动态热身
    - 夜间小腿抽筋 → 睡前补镁（200-400 mg 甘氨酸镁）+ 小腿拉伸

## 必备药物

| **药物** | **用途** | **注意事项** |
| --- | --- | --- |
| **对乙酰氨基酚**（泰诺） | 退烧、头痛、轻中度疼痛 | 每日 ≤3g；不与酒精同服（伤肝） |
| **布洛芬** | 退烧、头痛、炎性疼痛、痛经 | 饭后服用（伤胃）；外伤 48h 内慎用 |
| **口服补液盐（ORS）** | 腹泻/呕吐脱水补充 | 按说明冲配 |
| **蒙脱石散** | 急性腹泻吸附止泻 | 与其他药物间隔 2h 服用 |
| **氯雷他定**（开瑞坦） | 过敏（荨麻疹、花粉症、食物过敏轻症） | 不嗜睡型抗组胺 |
| **碘伏** | 伤口消毒 | 优于酒精和双氧水 |
| **创可贴 + 无菌纱布 + 医用胶带** | 伤口覆盖 | 定期检查有效期 |
| **弹性绷带** | 扭伤加压 | 不要缠太紧 |
| **体温计** | 发热监测 | 电子体温计最便捷 |
| **医用镊子 + 小剪刀** | 取异物/剪纱布 | 保持清洁 |

# 五、核心原则总结

<aside>
🎯

**记住这 7 条，覆盖 80% 的日常不适预防与急救逻辑。**

</aside>

### 预防层

1. **80% 的日常不适来自 4 个源头**：脱水、睡眠不足、久坐、饮食不当 → 先修复这 4 个再说其他
2. **身体的不适是信号，不是噪音** → 不要用止痛药/咖啡因掩盖信号，找到并解决根因
3. **最好的免疫力来自睡眠和运动**，不是保健品

### 急救层

1. **先排除红色信号** → 胸痛、意识改变、大量出血、呼吸困难 → 立即急诊，不犹豫
2. **脱水比你想的更危险** → 无论什么突发不适，补液几乎永远是对的
3. **炎症是修复的起点** → 外伤 48h 内不要急于消炎止痛（PEACE & LOVE > RICE）
4. **不确定时就就医** → 花几百块排除严重问题，永远比「再等等看」安全
