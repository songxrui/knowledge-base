你希望 GPT 帮你设计一套最优提示词，用来生成一个深度方案，帮助你：

你的深层目标不是“模仿 Dan Koe”，而是：

> 建立一个适合中国青年创作者的个人成长与内容商业操作系统。
>

1. **行为系统**：减少游戏和无意识刷媒体，恢复注意力与执行力。

5. **商业系统**：从内容影响力到服务、咨询、产品、课程或社群的路径。

# PART 2 — Problem Model

## Problem Category

这是一个复合型问题：

> **从内容消费者到内容创造者的身份转换问题。**
> 

更精确地说：

```
低执行力青年
→ 通过行为重塑恢复注意力
→ 用 AI 和 Notion 整理思想
→ 用写作训练输出内容
→ 在中国平台建立个人 IP
→ 逐步探索自由职业与一人商业模式
```

## Core Mechanism

你的成长闭环应是：

```
输入：Dan Koe / AI / 书籍 / 生活经验 / 健身 / 现实问题
        ↓
思考：用 AI 对话、Notion 卡片、问题拆解
        ↓
表达：写成短文、长文、脚本、观点
        ↓
发布：社交媒体持续公开输出
        ↓
反馈：阅读量、评论、收藏、私信、复盘
        ↓
产品化：服务、咨询、资料包、课程、社群、工具模板
        ↓
人生系统：形成你的世界观、能力栈和商业模式
```

## Key Variables

| 变量 | 当前状态 | 优化方向 |
| --- | --- | --- |
| 时间 | 每天 1 小时起步 | 先稳定，再增加 |
| 注意力 | 游戏、手机、刷媒体干扰大 | 行为设计 + 环境控制 |
| 输出 | 一直没发社交媒体 | 从低风险公开输出开始 |
| 写作 | 想像 Dan Koe 那样表达 | 中文化、平台化、受众化 |
| 知识库 | 已有 Notion 和 AI 对话 | 转为内容资产库 |
| 身体 | 饮食、健身已有复利行为 | 作为自律锚点迁移到写作 |
| 商业 | 想学 Dan Koe 模式 | 中国化个人 IP + 服务化起步 |
| 现金流 | 家庭存款 + 低门槛工作兜底 | 保障探索期安全感 |

## Optimization Targets

- 90 天内形成写作输出习惯；
- 从每天 1 小时启动，逐步提高到 2–3 小时深度创作；
- 至少发布 30–60 条内容；
- 建立 Notion 内容资产库；
- 形成 3–5 个稳定内容主题；
- 能把任意想法转化为一篇完整中文自媒体文章；
- 初步验证一个中国化个人 IP 方向；
- 游戏和刷手机时间显著下降；
- 从“学习 Dan Koe”变成“用自己的经历服务中国受众”。

## Failure Modes

1. **沉迷研究，不发布**
一直看 Dan Koe、整理 Notion、和 AI 对话，但不公开输出。
2. **过度追求完美**
想一开始就写出高质量文章，导致迟迟不发。
3. **照搬英文语境**
直接复制 Dan Koe 的表达、主题、商业模式，忽略中国平台受众。
4. **过度搭建系统**
Notion 越做越复杂，反而成为拖延工具。
5. **游戏和短视频吞噬注意力**
每天计划被即时快感打断。
6. **把个人 IP 理解成包装人设**
忽略真实能力、真实经历、真实解决方案。

## Generalized Solution Pattern

```
90 天个人 IP 启动模型：

第 1 阶段：清理注意力
减少游戏、短视频、无意识刷媒体。

第 2 阶段：建立最小写作习惯
每天 1 小时，完成一个想法卡片或短内容。

第 3 阶段：搭建知识资产库
把 AI 对话、书籍笔记、生活反思转成内容素材。

第 4 阶段：公开发布
从低压力平台开始，训练表达和反馈承受力。

第 5 阶段：形成内容飞轮
长文 → 短内容 → 选题库 → 系列内容 → 产品雏形。

第 6 阶段：探索商业化
服务、咨询、模板、资料包、训练营、社群。
```
