# 饮食 3.0

<aside>
🎯

**一句话目标**：低体脂表现 + 力量恢复 + 肠道稳定 + 皮肤改善 + 肾保护。

**三个旋钮**：蛋白每天稳定 · 碳水按训练强度调 · 菌群靠多样性不靠极端。

**80/20 规则**：80% 热量来自高分食物，20% 留给任何你想吃的（防崩溃安全阀）。

## 0️⃣ 当前主链定位

### 减脂优先级

- **睡眠优先级最高**：睡不好时，同等热量缺口下更容易掉肌肉、难掉脂肪
- **热量缺口是本质**：但优先用结构性减法，不用极端压热量
- **蛋白稳定**：每天 ≥130g，优先保护恢复与瘦体重
- **力量训练不降重量**：减容量可以，核心动作重量不主动下调
- **低强度有氧作为加速器**：优先快走 / Zone 2，而不是高压力 HIIT
- **防代谢适应**：保留每周 1 次高碳日，不做长期激进赤字

# ④ 训练日食谱模板

**变体规则：**

- **爬坡日**：午餐米饭 → 150g；取消训练后补碳
- **休息日**：早餐麸皮 → 40g；午餐去米饭；坚果 +10g
- **高碳日（1/周）**：午餐米饭 → 400g；练后红薯 → 80g

### ☀️ 早餐

- 燕麦麸皮 60g
- 鸡蛋 3 个
- 酸奶 200g + 奇亚籽 10g
- 抹茶/可可 5g
- **菊粉** 5–8g（上限 10g）
- 🆕 **柠檬水**一杯（肾保护）
- 间隔车前子壳 1h+
- 多维 + 鱼油 + D3 + B12

### 🍚 午餐

- 米饭 250g
- 鹰嘴豆 100g
- 橄榄油 10g
- 蔬菜 ≥200g
- 🆕 **番茄**入菜（配油烹熟）
- 🆕 **大蒜** 2–3 瓣随菜
- 🆕 **陈醋**淋菜
- 黑枚杵原浆 20–30ml

### 🏋️ 训练前后

**训练前：**

- 乳清 30g
- 肌酸 5g

**训练后：**

- 米饭 200g *或* 红薯片 60g
- 乳清 30g

### 🌙 晚餐

- 黄豆 100g
- 坚果 25g
- 黑芝麻粉 10g
- 洋车前子壳 5g
- 🆕 **菌菇**（平菇/杏鲍菇）随菜
- 🆕 **紫菜蛋花汤** 或 **凉拌海带丝**
- 🆕 **姜丝**随菜
- 甘氨酸镁 + Zn + Q10 + 花青素 + 虾青素
- 姜黄粉 1–3g 配黑胡椒 + 脂肪

# ⑥ 菌群策略

- **每天**：新增 ≥1 种蔬菜轮换
- **每周**：≥30 种植物；抗性淉粉 3 次（冷米/冷薯）；发酵食品 ≥4 次
- **目的**：炎症↓ 外貌稳定↑（Evidence：中高）

---

# ⑦ 肠道专项管理（直肠炎 + 痔疮史）

- **纤维渐进**：菊粉从 5g 起，每周增 2g，上限 10g
- **洋车前子壳**：与菊粉间隔 ≥1h（可溶性凝胶 vs 益生元发酵，机制不同）
- **高纤维必须配水**：否则反而加重便秘
- **久坐中断**：每 30–45min 起身 → 减少肛周静脉压力
- **触发物管理**：西兰花暂时避开，十字花科用**花菜/包菜**替代
- **甘氨酸修复**：每周 2–3 次骨汤 → 修复肠黏膜层

---

# ⑧ 痤疮专项管理

- **低 GI 策略**：白米饭部分替换红薯/糙米/紫薯（RCT：低 GI 12 周 → 痤疮病灶 ~−20%）
- **乳制品实验**：先 4 周减少牛奶，保留酸奶，观察变化（发酵乳制品关联弱于鲜奶）
- **Omega-3**：已在补鱼油 → 保持；加上沙丁鱼整鱼加强
- **锌**：已在补 → 保持
- **糖控制**：奶茶 ≤2 杯/周，选少糖/无糖
- **血糖三重削峰**：进食顺序（先菜后碳）+ 醋淋菜 + 餐后 10min 散步

---

# ⑨ 肾保护专项

- **水 ≥2L/天**：目标清澈尿液
- **柠檬水每日**：柠檬酸盐碱化尿液，降低肾结石风险
- **控钠**：<2300mg/天；辣鱼仔、外卖是主要钠源 → 计入 20% 额度
- **不乱吃药**：尤其是止痛药和不明中药
- **每年验一次尿和血**：关注胱抑素 C 和尿蛋白
