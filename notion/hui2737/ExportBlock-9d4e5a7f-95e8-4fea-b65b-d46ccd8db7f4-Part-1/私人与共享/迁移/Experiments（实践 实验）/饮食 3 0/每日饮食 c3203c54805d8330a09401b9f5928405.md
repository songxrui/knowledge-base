# 每日饮食

<aside>
✅

这页只做一件事：把你的 **饮食（Plant-forward × 地中海）+ 肠道/痔友好 + 助眠** 压成一张「每天只打勾」的执行表。

规则：**先打卡，再优化；先稳定 7 天，再改变量。**

</aside>

## 边界

- **不吃/强不耐受**：番茄（会恶心/曾呕吐）、西兰花（曾触发肠道不适）。
- **乳糖代谢弱**：乳制品优先 **无糖酸奶**；牛奶/奶茶按耐受降频，晚上尽量不喝。
- **咖啡因慢代谢**：咖啡/奶茶/茶/抹茶/可可尽量都放在 **12:00 前**。
- **痔疮优先级高于“吃得完美”**：任何明显刺激排便、加重痛胀/便血的吃法，直接降级或删除。

## ✅ 每日必做（E0/E1）

### 1）蛋白（E0）

- [ ]  **蛋白 ≥130g**（训练/减脂期硬约束）
    - 最低实现：鸡蛋 + 乳清 + 豆类（黄豆/鹰嘴豆/黑纳豆）

### 2）发酵食品 1 份（E0）

- [ ]  **无糖酸奶 200g**（默认）
- [ ]  黑纳豆（可选，建议每周 3–5 次，不必天天）

### 3）蔬菜保底（E0）

- [ ]  **蒸一锅蔬菜**（花菜/包菜/油麦菜/白菜/菠菜/莴笋 任选 1–2 种）
    - 当天蔬菜没吃：用「纤维兜底」（见下方），但第二天须补回

### 4）水果 1–2 份（E1）

- [ ]  苹果（默认）
- [ ]  香蕉（更适合训练前后；休息日控制量）
- [ ]  芒果（“情绪吃”不天天）

### 5）补剂三段（E1）

- [ ]  早餐：多维 + 鱼油 + D3 + B12
- [ ]  训练前：乳清 + 肌酸
- [ ]  晚餐：甘氨酸镁 + Zn + Q10 + 花青素 + 虾青素

### 6）肠道/痔（E1）

- [ ]  洋车前子壳 **5g**（配足水）
- [ ]  菊粉 **3–8g**（从 5g 起；与洋车前子壳间隔 ≥1 小时）

> 说明：纤维/益生元只能兜底「纤维与菌群底物」，不能长期替代蔬菜本体。
> 

## 🍽 懒人模板

### 模板 A｜蒸蛋机“一锅流”（推荐默认）

- [ ]  下层：花菜或包菜 + 叶菜（油麦菜/白菜/菠菜）
- [ ]  上层：蒸蛋（鸡蛋羹/水煮蛋）
- [ ]  出锅加分：香菜 + 少量酱油/醋（提升可吃性）

### 模板 B｜外卖日（不内疚，只控结构）

- [ ]  **强制两份蔬菜**：清炒油麦菜/白菜/菠菜 + 花菜/包菜
- [ ]  主菜优先：鸡蛋/豆腐/鱼/瘦肉
- [ ]  避免：高糖醋/重油/麻辣爆炒

## 🔁 日类型调参（只调主食/碳水）

- [ ]  训练日：碳水集中在训练前后（米饭/红薯/燕麦）
- [ ]  爬坡日：午餐主食减量（米饭约 150g 级别）
- [ ]  休息日：主食前置 + 晚餐低碳
- [ ]  高碳日：每周 1 次；高碳尽量低油低糖

## ⚡ 加速器（E2，可选）

- [ ]  抗性淀粉 1 份（冷米/冷薯，周内 ≥3 次）
- [ ]  发酵食品额外 +1（酸奶以外的发酵食物，按耐受）
- [ ]  今日无高钠零食/外卖（控浮肿）
- [ ]  胶原触发：明胶粉 10g 或骨汤 1 碗（周内 2–3 次）
