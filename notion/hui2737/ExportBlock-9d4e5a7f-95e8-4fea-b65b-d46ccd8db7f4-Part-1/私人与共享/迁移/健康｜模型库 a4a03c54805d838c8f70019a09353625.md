# 一、总公式

> **健康存量 = 睡眠 × 训练 × 饮食 × 情绪稳定 × 时间。** 缺一个变量，其他乘起来都接近 0。
> 

顺序硬规则：**睡眠 > 训练 > 饮食 > 情绪 > 信息 > 输出。** 上游没修好，不动下游。

# 二、六个底层变量（万病同源）

几乎所有重大慢性病（心血管、癌、糖尿病、阿尔茨海默、自免）共用同一组上游驱动。控制这 6 个 = 同时降低数十种疾病概率。

| **变量** | **坏在哪** | **最大杠杆动作** |
| --- | --- | --- |
| 慢性低度炎症 | 万病之源（CRP↑→心血管 / 癌 / 抑郁 / 衰老加速） | 降内脏脂肪 + 减 UPF + Omega-3 + 睡眠 |
| 胰岛素敏感性 | 抵抗 → 代谢综合征、脂肪肝、心血管、部分癌 | 力量训练 + 餐后散步 + 控精制碳水 |
| 血管健康（ApoB） | 动脉斑块从青年开始累积，21 岁是窗口期 | 知道自己 ApoB 和 Lp(a) + Zone 2 + 不抽烟 |
| 睡眠 | ＜6h → 全因死亡 +13%、癌 +40%、心血管 +48% | 固定起床时间 > 固定入睡；卧室 18–20°C 全黑 |
| 肠道屏障 | 肠漏 → LPS 入血 → 全身炎症 / 自免 / 情绪 | 30 种植物/周 + 发酵食品 + 少酒精少 NSAIDs |
| 慢性压力 | HPA 失调 → 皮质醇高位 → 免疫抑 / 海马萎缩 | 晨光 + 有氧 + 面对面社交 + 生理性叹息 |

# 三、激素跷跷板

不要单看一个激素，看比值和节律。

- **皮质醇 vs 睾酮**：跷跷板。慢性压力 / 缺睡 → 皮质醇↑ → 睾酮↓。
- **胰岛素 vs 生长激素**：吃饭 → 胰岛素↑ → GH 被抑制。深睡 + 空腹 → GH 脉冲。睡前 3h 不进食是这条机制的应用。
- **皮质醇 vs 褪黑素**：晨高夜低 / 夜高晨低。被光开关。错位即节律崩。
- **多巴胺基线 vs 峰值**：每次大峰之后基线下跌。短视频/色情/游戏让基线塌方。运动 / 冷暴露 / 深度专注是少数能**抬高基线**的活动。

# 四、昼夜节律：唯一的主时钟

光信号 → ipRGC → SCN → 全身外周时钟。

仅靠两件事：**固定起床时间 + 醒后 30 分钟内 10 分钟户外光**，可改善睡眠质量 40–60%、白天精力 30–50%。这是单位投入回报最高的健康干预。

打乱节律的三件事：晚睡、不见晨光、深夜屏幕。

# 五、自主神经：迷走神经张力

现代人默认交感过激活。恢复力 = 副交感切回的速度，可量化为迷走神经张力 / HRV。

- **即时切回**：生理性叹息（双吸一长呼）×3 次；延长呼气（吸 4 / 呼 8）；冷水拍脸；哼鸣。
- **长期增强**：Zone 2 有氧、冷暴露、每日冥想 8 周、面对面社交、深睡。

# 六、多巴胺基线模型

目标不是禁欲，是恢复对**普通快乐**的敏感度。

| 类型 | 峰值倍数 | 对基线 |
| --- | --- | --- |
| 运动 / 冷水 / 深度工作 | 1.5–2.5× | **抬升基线** |
| 美食 / 音乐 / 一般社交 | 1.5–2× | 轻降，快恢复 |
| 短视频 / 游戏 / 色情 / 尼古丁 | 2–3× 间歇式 | **显著塌基线** |

推论：**不是这些事不让人快乐，而是它们让你之后什么都不快乐。**

# 七、肌肥大与训练核心机制

机制权重已被高质量元分析重排：

1. **机械张力（首要）**：mTORC1 通路被高负荷激活。决定性。
2. **代谢压力（辅助）**：高次数 + 短间歇 + 高泵感。次要。
3. **肌肉损伤（混杂）**：酸痛 ≠ 增肌；过度损伤反而拖恢复。

核心训练变量（循证剂量）：

- 训练量：**每肌群 10–20 组/周**
- 频率：每肌群 **2 次/周**（量相同时频率本身影响小，但分两天能保证后半组质量）
- 负荷：30%–85% 1RM 都行，关键是 **RIR 1–3**
- 组间休息：复合动作 **2–3 分钟**（让 PCr 恢复）
- 同期训练干扰：跑步对力量的干扰 > 骑行；间隔 ≥6h

# 八、能量系统与有氧分区

| 系统 | 时长 | 用途 |
| --- | --- | --- |
| 磷酸原（PCr） | 0–10s | 最大力量、爆发力，组间休息 3–5 分钟 |
| 糖酵解 | 30–120s | 肌肥大主区间，代谢压力来源 |
| 有氧 Zone 2 | ＞30 分钟 | 线粒体密度 / 脂肪氧化 / 代谢灵活性 |
| 有氧 Zone 5 / HIIT | 3–8 分钟段 | VO₂max 提升（最强死亡率预测指标） |

**VO₂max 从同龄人最低 25% 到最高 2.5%，全因死亡风险降低 5×。** 比戒烟和治糖尿病的杠杆都大。

# 九、mTOR / AMPK 开关

身体只有两种状态：**建设（mTOR）vs 修复（AMPK / 自噬）**。两者要轮替，不能永远 mTOR 在线。

- 进食 + 蛋白 → mTOR↑ → 肌肉合成
- 空腹 + 运动 + 冷暴露 → AMPK↑ → 自噬清除受损细胞

应用：进食窗口 10–12h；训后 30 分钟蛋白窗口；不要从早到晚不停吃零食。

# 十、饮食 80/20 机制层

决定体脂与代谢的因子按权重排序：

1. **总热量**（决定体脂升降）
2. **蛋白质 ≥1.6 g/kg**（保肌、饱腹、恢复）
3. **膳食纤维 25–35 g**（菌群 / 血糖 / 饱腹）
4. **UPF 占比**（越高越容易过量、越扰乱奖励系统）
5. **游离糖 / 液体糖**（甜饮、奶茶；不顶饿、伤牙、推高总热量）

血糖三重削峰：**先菜→蛋白→碳水 + 醋 + 餐后散步 10 分钟。** 餐后血糖 AUC 降 20–40%。

# 十一、福格行为模型 B = M × A × P

行为 = 动机 × 能力 × 提示。

- **坏行为**：删提示 + 增摩擦 + 延迟 + 替代。手机不进床、奶茶不囤家。
- **好行为**：降门槛 + 绑场景 + 给反馈。运动鞋放门口、晨光配早咖啡。

冲动急救：停 10 分钟 → 命名冲动 → 定位（饿/累/烦/孤独/压力/无聊）→ 走 3 分钟 / 喝水 / 写触发。

# 十二、衰退时间表（不干预的默认轨迹）

| 系统 | 起点 | 速度 | 能不能逆 |
| --- | --- | --- | --- |
| VO₂max | 25 岁 | −10%/十年 | Zone 2 可逆 10–20 年 |
| 肌肉量 | 30 岁 | −3–5%/十年（不练 −8%） | 力量训练可维持到 70 岁 |
| 骨密度 | 30 岁 | −0.5–1%/年 | 30 岁前的存量决定终身 |
| 睾酮 | 30 岁 | −1–2%/年 | 睡眠 + 力量 + 低体脂可显著延缓 |
| 端粒 | 持续 | 受生活方式强烈影响 | 运动+睡眠可减缓 40–50% |

推论：**21 岁不是「还年轻」，是「红利窗口」。** 这一阶段每分投入 ROI 最高，30 岁后只能延缓不能再造。

# 十三、中国男性疾病图谱（按可预防度排序）

潜伏期长 → 越早干预越便宜。

- 代谢综合征 / 2 型糖尿病：**90%+ 可预防**
- 心血管：**80%+ 可预防**（ApoB 是核心指标，不是 LDL-C）
- 慢性肾病 / COPD：**70%+ 可预防**
- 肝癌（乙肝 + 酒精）：**疫苗 + 不饮酒 = 终身收益**
- 胃癌（幽门螺杆菌）：**一次呼气试验 + 一次根除即可**
- 结直肠癌：**纤维 + 控加工肉 + 45 岁起肠镜**
- 阿尔茨海默：症状前 20–30 年开始，深睡 + 有氧 + 社交 + 控血管风险是仅有的延缓手段

# 十四、干预优先级（80% 收益的 5 件事）

几乎不花钱，撬动 80% 生物系统。

1. 睡 7–9h + 固定起床
2. 醒后 10 分钟晨光
3. 运动 30–60 分钟（力量 + Zone 2）
4. 进食窗口 10–12h + 蛋白达标 + 减 UPF
5. 面对面社交 + 每日呼吸练习

剩下 20%（补剂、冷暴露、桑拿、护肤进阶、高级筛查）是锦上添花。**先打地基，再装修。**

# 十五、调参原则（系统稳定性）

- **一次只动一个变量**，观察 1–2 周。多变量同改 = 无法归因 = 等于没实验。
- **状态变差先查 4 件事**：睡眠、日照、活动量、刺激摄入。不要先怀疑意志力。
- **不堆动作，先做减法**：先停损（熬夜 / 短视频 / 奶茶 / 油炸 / 冲动消费 / 无问题输入），再加优化。
- **20% 自由额度**：留给情绪食物 / 快乐活动，否则系统反弹。
- **指标 <3/5 持续两周 → 降级该变量**，不硬扛。

# 十六、红线触发降级（不讨论，自动执行）

- 睡眠 ＜6h 连续 3 天 → 砍训练量 30–40%
- TMJ / 关节痛 ≥3/10 持续 3 天 → 该动作减重 10–20% 或替换
- HRV 持续走低 + 静息心率持续走高 → 增加休息日
- 性欲 / 动力骤降 → 查睡眠 + 总训练量 + 热量缺口
- 持续动力缺失 + 社交回避 → 系统降级，先减承诺再谈思考
