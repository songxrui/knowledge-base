# 【塔勒布宝典】专题三：风险共担与否定法（减法智慧）

Anti-Use: 需要快速发散创造/探索时，不要用过强的否定法把试错空间压没；先限定范围再过滤。
Created: 2026年5月15日 12:15
Failure Modes: 只听观点不看代价；加法上瘾导致系统膨胀；把否定法写成哲学不落清单；用新奇信息喂自己导致噪声；把“名气”当可靠性。
Last Verified: 2026年5月9日
Model Type: Meta
Reuse Count: 0
Steps: 1) 问 Skin in the Game：对方承担什么代价？
2) 先做减法：列“不做清单”并写成规则
3) 用 Lindy 过滤信息：优先长期存活的经典
4) 把规则落到日历/环境/权限（能执行）
When to Use: 用于识人/过滤建议（尤其是投资、医疗、职业建议）；用于把健康/执行/风险管理做成“减法防火墙”；用于信息输入白名单。

<aside>
⚔️

**塔勒布·过滤网卡**：1) 没有风险共担就不可信；2) 生活与系统优化优先做减法。

</aside>

| 工具 | 一句话 | 落地问法/动作 |
| --- | --- | --- |
| Skin in the Game | 不承担代价的人，不该拥有影响力 | "你押了什么？错了你损失什么？"（钱/名誉/职业/健康） |
| Via Negativa（否定法） | 我们更确定“什么有害”，所以先删掉有害项 | 先写 **不做清单**：熬夜/高糖/刷短视频/无止损下注…并写成可执行防火墙 |
| Lindy 效应 | 对不“自然死亡”的思想：活得越久，越可能继续存活 | 信息输入优先经典与长期存活做法；新奇观点先降权 |

<aside>
✅

**执行准则**：先问代价（共担）→ 再做减法（防火墙）→ 最后才谈加法（优化）。
