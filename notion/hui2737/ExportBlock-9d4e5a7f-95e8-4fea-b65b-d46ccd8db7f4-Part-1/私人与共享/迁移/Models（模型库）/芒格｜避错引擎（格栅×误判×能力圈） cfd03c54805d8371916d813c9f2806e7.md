## ① 一句话模型

复杂决策先防蠢：至少两个不同模型同向，且通过误判体检，才允许下注。 [B]

## ② 决策三件套

- **目标函数**：这件事真正优化什么？≤30 字。 [B]
- **出局线**：什么情况会造成归零、失信、健康损伤或长期后悔？ [A]
- **可观测变量**：用 3 个指标判断进展；没有指标则降级为探索。 [B]

## ③ 格栅透镜

| 透镜 | 核心问句 | 动作 |
| --- | --- | --- |
| 激励 | 谁被奖励？奖励把行为推向哪？ | 建议者不担责则降权 |
| 复利 | 5 年后还累积吗？ | 不复利则不做主线 |
| 概率 | 赔率、胜率、仓位各是多少？ | 超仓则缩小 |
| 机会成本 | 同时间更好选择是什么？ | 答不出先不做 |
| 安全边际 | 条件差 30% 还成立吗？ | 不成立则降规模 |

## ④ 八个误判红灯

- 奖惩超级反应：短期甜头压过长期代价 → 写长期代价。 [B]
- 避免不一致：为旧承诺硬撑 → 问“第一次遇到还会选吗？” [B]
- 社会认同：大家都做我也做 → 问“只有我做还愿意吗？” [B]
- 被剥夺反应：亏了更想加码 → 只看从今天起的边际收益。 [A]
- 权威影响：头衔替代证据 → 问“证据是什么？承担什么代价？” [B]
- 对比错误：渐进恶化不自知 → 问“再坏 20% 是否出局？” [B]
- 嫉妒驱动：追排名非价值 → 问“愿 100% 交换人生吗？” [C]
- Lollapalooza：多 bug 同向 → 强制 24 h 冷却。 [B]

## ⑤ 能力圈与出手规则

1. 圈内：懂变量、懂风险、懂退出；可集中。 [B]
2. 圈边：小注试错，只买信息。 [B]
3. 圈外：默认不做；若必须做，找担责者校验。 [B]
4. 出手条件：赔率高 + 能力圈内 + 安全边际足 + 熔断明确。 [B]

## ⑥ if-then 卡

- if 决策牵涉钱/关系/职业/健康 → 写三件套；不写不做。 [B]
- if 情绪强度 >7/10 → 24 h 后重看。 [B]
- if 只有一个模型支持 → 降级为小实验。 [B]
- if 沉没成本牵引 → 旧成本归零，只看边际收益。 [A]
- if 需要包装事实才成立 → 停止；改写为真实约束。 [B]
