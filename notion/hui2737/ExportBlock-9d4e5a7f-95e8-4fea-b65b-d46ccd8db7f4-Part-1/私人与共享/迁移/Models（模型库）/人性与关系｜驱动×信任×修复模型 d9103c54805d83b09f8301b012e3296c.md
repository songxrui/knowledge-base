<aside>
🧬

**人性与关系模型**：行为不是先判好坏，而是先看驱动；关系不是先争对错，而是看信任、情绪连接和修复能力。

</aside>

## ① 一句话模型

关系质量 = 驱动识别 × 信任一致性 × 情绪回应 × 沟通质量 × 修复能力。 [B]

## ② 五个行为驱动

| 驱动 | 底层问题 | 误判 | 杠杆 |
| --- | --- | --- | --- |
| 欲望 | 我想要什么？ | 把想要当喜欢 | 拆成可兑现小奖励 |
| 恐惧 | 我会失去什么？ | 把回避当懒 | 降低威胁，增加可控感 |
| 奖励 | 什么值得重复？ | 意志力硬拼 | 改环境、门槛、即时反馈 |
| 地位 | 我排第几？ | 观点冲突当真相冲突 | 退出低质排名 |
| 归属 | 我会被接纳吗？ | 迎合当成熟 | 真实+边界+可修复 |

## ③ 关系六系统

| 系统 | 核心问题 | 失衡信号 | 最小修复 |
| --- | --- | --- | --- |
| 吸引力 | 我为何靠近？ | 把稀缺/征服感当喜欢 | 生理/情绪/长期匹配分开打分 |
| 信任 | 我能交出脆弱吗？ | 承诺不兑现、高警觉 | 记录兑现率和一致行为 |
| 依恋 | 亲密让我安心还是失控？ | 追逃循环、过度确认 | 先调节，再沟通 |
| 情绪连接 | 我是否被感受到？ | 只讲道理，内里疏离 | 先回应感受，再谈问题 |
| 沟通质量 | 问题被处理还是升级？ | 指责、读心、防御 | 事实→感受→需求 |
| 修复稳定 | 出问题能不能修？ | 同一问题反复复发 | 看尊重、承担、行为修正 |

## ④ 诊断流程

1. 写行为，不猜动机：对方/我实际做了什么？ [B]
2. 判驱动：它满足了哪 1–2 个系统？ [B]
3. 写隐藏收益：短期获得什么止痛或地位？ [B]
4. 写长期代价：健康、信任、自由、时间损失是什么？ [B]
5. 选最小动作：边界、回应、离场、修复四选一。 [C]

## ⑤ 红线

- 暴力、胁迫、持续撒谎、冷暴力、羞辱控制 → 优先离场与求助，不做沟通技巧实验。 [A]
- 自伤念头或持续低落 ≥14 天 → 专业支持。 [A]
- 信任破裂后只有道歉，没有连续行为证据 → 不恢复原级别。 [B]
- 长期耗损大于滋养 → 降级关系。 [C]

## ⑥ if-then 卡

- if 我想立刻反击 → 先问：我在保护事实还是保护地位？ [C]
- if 对方只讲道理 → 先要情绪回应，再谈方案。 [C]
- if 我开始讨好 → 写我害怕失去什么。 [C]
- if 冲突反复出现 → 不看道歉，看下一周行为。 [B]
- if 上头很强但不稳定 → 降权为刺激，不当长期适配。 [C]
