# 【芒格宝典】专题二：人类误判心理学与25个Bug

Anti-Use: 纯创造/发散阶段不必过度体检；避免把一切都当 bug 导致自我否定。
Created: 2026年5月15日 12:15
Failure Modes: 只背概念不做体检；情绪激活时仍做重大决定；只找支持证据；把沉没成本当理由；被权威/群体带节奏；不写熔断导致越错越大。
Last Verified: 2026年5月9日
Model Type: Decision
Reuse Count: 0
Steps: 1) 先写决策三件套：目标函数/出局线/可观测变量
2) 过“误判体检”：奖励惩罚、避免不一致、社会认同、被剥夺、权威、对比、嫉妒、Lollapalooza
3) 每条写一个反证问题：若为真则不做
4) 把熔断写成动作：触发即停/撤/降级
5) 事后复盘：到底是哪个 bug 触发
When to Use: 用于重大决策/交易/关系与职业选择前的“反误判体检”：把常见心理 bug 显式化，强制写反证与熔断，避免在奖励/从众/权威/沉没成本里翻车。

<aside>
🧠

**芒格·误判体检卡**：不是让你更聪明，而是持续避免变蠢（尤其在奖励与情绪被点燃时）。

</aside>

| 高危 bug | 典型表现 | 当场反证问句（写下来） |
| --- | --- | --- |
| 奖励/惩罚超级反应 | 为了短期好处牺牲长期 | "谁在被激励？激励会把行为推向哪里？" |
| 避免不一致 | 死守旧结论与身份，不愿修正 | "如果我是第一次遇到这事，我还会这么选吗？" |
| 社会认同 | 大家都做→我也做 | "如果只有我一个人做这个选择，我还愿意吗？" |
| 被剥夺超级反应 | 为了不失去而做极端决策（沉没成本） | "我是在做更优选择，还是在避免承认亏损？" |
| 权威影响 | 看到头衔就投降 | "证据是什么？他/她承担什么代价？" |
| 对比错误 | 温水煮青蛙，渐进恶化不自知 | "如果这事再坏 20%，我还扛得住吗？" |
| 嫉妒 | 为赢地位游戏而决策 | "我是在追价值，还是在追排名？" |
| Lollapalooza | 多个 bug 叠加→极端行为 | "此刻是否多股力量同向推我冲动？先冷却再决定。" |

<aside>
✅

**用法**：重大决策/开仓前，把以上反证问句写完再做（不写不做）。

</aside>
