## ① 一句话模型

社会力量 = 共同想象 × 制度执行 × 注意力控制；自由来自看见叙事。 [B]

## ② 共同想象

- 国家、公司、金钱、品牌、身份，都是能组织大规模协作的故事。 [B]
- 故事不是虚假无用；它能产生真实约束和真实资源。 [B]
- 个人 IP 本质是小型共同想象：价值观、承诺、符号、信任。 [C]

## ③ 进步陷阱

- 奢侈品会变成必需品，必需品会变成新义务。 [B]
- 现代小麦：房贷、短视频、工作群、消费身份、虚荣社交。 [C]
- 判断句：它最初承诺解放我，现在是否开始奴役我？ [C]

## ④ 可骇入动物

- 算法结合生物学知识、算力和数据，会比你更懂你的弱点。 [B]
- 推荐流不是中立信息源，是注意力捕获机器。 [B]
- 内观的功能不是玄学放松，而是更快识别欲望、恐惧和逃避。 [C]

## ⑤ 防火墙

1. 手机物理隔离，尤其床边和 P0 前。 [C-self]
2. 信息流只在固定窗口，窗口外不打开。 [C-self]
3. 每天 60 秒问：我现在在逃避什么？ [C]
4. 睡前低刺激：书、日记、放空，不刷算法。 [C]
5. 每周删 1 个劫持源或降权限。 [C-self]

## ⑥ if-then 卡

- if 一个故事要求牺牲健康/自由/诚实 → 降级审查。 [C]
- if 工具变成义务 → 重设边界或删除。 [C]
- if 刷信息流 >30 min → 记录触发并物理隔离。 [C-self]
- if 品牌叙事无交付 → 视为空壳。 [C]
- if 反技术情绪上头 → 改成规则化使用，不做道德批判。 [C]
