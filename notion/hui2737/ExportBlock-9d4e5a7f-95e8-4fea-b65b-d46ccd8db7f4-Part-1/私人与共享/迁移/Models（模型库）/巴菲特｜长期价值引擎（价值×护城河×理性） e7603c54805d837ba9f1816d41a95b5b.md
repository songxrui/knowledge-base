🏰

**巴菲特｜长期价值引擎**：价格是市场情绪，价值是资产本身。只买看得懂、有护城河、有安全边际、能长期复利的东西。

## ① 一句话模型

长期回报 = 内在价值 × 护城河 × 安全边际 × 情绪稳定 × 时间。 [B]

## ② 价格 vs 价值

- 价格：市场先生今天给的报价，情绪驱动。 [B]
- 价值：资产未来现金流、能力、声誉、复利潜力。 [B]
- 动作：若不能估算价值，只能观察，不能重仓。 [B]

## ③ 四类护城河

| 护城河 | 企业例子 | 个人迁移 |
| --- | --- | --- |
| 品牌 | 信任溢价 | 稳定观点和交付信誉 |
| 转换成本 | 迁移麻烦 | 你的模板/系统嵌入对方流程 |
| 网络效应 | 用户越多越有价值 | 公开输出带来协作与反馈 |
| 低成本优势 | 规模/流程优势 | AI+模板降低产出成本 |

## ④ 安全边际

- 买入价必须显著低于估算价值；估算错了仍不死。 [B]
- 个人系统中，现金、健康、时间冗余就是安全边际。 [B]
- 交易/项目中，单次失败必须小到不影响下一次出手。 [A]

## ⑤ 击球区

- 能力圈边界比能力圈大小更重要。 [B]
- 好球不到，不挥棒；多数机会不需要反应。 [B]
- 只在“看得懂 + 价格好 + 护城河深 + 安全边际足”时重注。 [B]

## ⑥ if-then 卡

- if 只因涨跌心动 → 先估值，不估不买。 [B]
- if 看不懂商业模式 → 不碰；或小仓买信息。 [B]
- if 没有护城河 → 不做长期主线。 [B]
- if 市场情绪极端 → 问它是在服务我，还是指挥我。 [B]
- if 一项技能无法滚雪球 → 降级为工具，不做主线。 [C]
