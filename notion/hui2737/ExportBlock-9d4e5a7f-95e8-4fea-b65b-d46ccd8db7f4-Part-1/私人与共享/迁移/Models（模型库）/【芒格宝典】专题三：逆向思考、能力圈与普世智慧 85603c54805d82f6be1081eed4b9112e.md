# 【芒格宝典】专题三：逆向思考、能力圈与普世智慧

Anti-Use: 需要快速迭代的试错场景不必过度“等待”；但仍需保留出局线，避免一次性归零。
Created: 2026年5月15日 12:15
Failure Modes: 只想如何赢不想如何不死；跨圈击球；频繁操作导致摩擦与错误累积；把耐心当懒散而不是筛选；用谎言维持人设导致成本爆炸。
Last Verified: 2026年5月9日
Model Type: Meta
Reuse Count: 0
Steps: 1) Invert：先列“会让我破产/崩掉/后悔”的死法清单
2) 能力圈：明确我懂什么/不懂什么（边界写出来）
3) 设出手条件：赔率高 + 在能力圈 + 有安全边际
4) 其他时间：学习/观察/等待（不乱动）
5) 诚实原则：用低成本真实减少系统摩擦
When to Use: 用于投资/职业/项目的长期策略：用逆向思考先避开“死法”，用能力圈限定可下注范围，用极端耐心只在高赔率时出手。

<aside>
🛡️

**芒格·普世行动卡**：Invert（先别死） + 能力圈（别跨界） + 耐心（只在高赔率出手）。

</aside>

| 原则 | 一句话 | 落地动作 |
| --- | --- | --- |
| Invert | 很多问题正向无解，先反过来列“死法” | 写“悲惨处方/破产处方”清单，并设成日常防火墙 |
| 能力圈 | 不知道边界=没有能力圈 | 写：我懂什么/不懂什么；不懂的默认不下注 |
| 极端耐心 | 不频繁操作；只在高赔率且安全边际足时重注 | 设“出手条件”模板：赔率/安全边际/出局线；不满足就等待 |
| 诚实=低成本 | 说真话减少维护成本 | 把关键事实写清楚：不包装、不自欺；用记录替代辩解 |

<aside>
✅

**一句话**：先把“会死的路”删掉，再把“高赔率的路”留下。

</aside>
