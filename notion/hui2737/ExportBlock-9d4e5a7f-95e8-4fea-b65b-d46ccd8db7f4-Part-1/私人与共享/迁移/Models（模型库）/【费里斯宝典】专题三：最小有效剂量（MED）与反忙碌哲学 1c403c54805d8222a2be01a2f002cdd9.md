# 【费里斯宝典】专题三：最小有效剂量（MED）与反忙碌哲学

Created: 2026年5月15日 12:15
Model Type: Learning
Reuse Count: 0

【材料识别】方法论型为主（目标→最小路径→停止规则），次为理论框架型（投入-结果临界点/“沸点”抽象）。

【核心概念】MED=达到达成信号所需最小投入集合；达成信号/沸点=1个可观测指标阈值；最小路径=≤2个关键动作；Stop Rule=达标即停并转投下一瓶颈；反忙碌=识别是否在回避关键动作。

【机制链】(1) 信号明确→只做关键动作→更快达标→停止加码→资源转投；(2) 信号缺失→无限加码→忙碌上瘾→系统过载→目标不推进。

【步骤（目标函数/DoD）】1写达成信号：把愿望变验收；DoD=指标+阈值+观察窗口。2设最小路径：最大化单位时间推进；DoD=动作≤2且写频率/时长。3设Stop Rule：防努力漂移；DoD=达标即停+写明转投目标。4每日30秒检查：反回避；DoD=两问+一句行动承诺。

【隐含假设】目标可被少数指标代表；动作与指标可观察关联；你愿意用规则克服“继续加码”的冲动。

【边界】硬门槛/合规不可跳步任务：MED=门槛本身。

【迁移】内容（周X卡片/1长文）；训练（周3次主项+RIR区间）；交易（周复盘N笔+偏差清单，防过度看盘）。

【薄弱点】把MED当偷懒；选虚指标（时长/页数）；无Stop Rule导致过载。

【3句总结】先定达成信号，再只做最小路径。达标立刻停止加码并转投瓶颈。无信号的努力=忙碌上瘾。
