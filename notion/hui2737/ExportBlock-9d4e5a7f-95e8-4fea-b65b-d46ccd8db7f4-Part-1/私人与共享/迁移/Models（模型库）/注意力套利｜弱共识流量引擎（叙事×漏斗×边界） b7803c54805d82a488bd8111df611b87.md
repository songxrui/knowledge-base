## ① 一句话模型

注意力资产 = 弱共识窗口 × 强钩子 × 证据承接 × 合法边界 × 转化漏斗。 [C]

## ② 弱共识套利

- 弱共识：大众未理解、未定价、规则仍在形成的窗口。 [C]
- 价值来自速度和信息差，风险来自规则不稳。 [C]
- 若无法写清合法边界，默认不做。 [C]

## ③ 注意力漏斗

| 层级 | 指标 | 目标 |
| --- | --- | --- |
| 曝光 | 浏览/搜索/讨论 | 进入视野 |
| 信任 | 收藏/评论/关注 | 证明不是噱头 |
| 资产 | 私域/模板/案例/产品 | 沉淀可复用价值 |
| 转化 | 付费/合作/复购 | 验证商业闭环 |

## ④ 合法版操作

1. 强钩子：反常识但不造假。 [C]
2. 证据块：事实、案例、边界分开写。 [C]
3. 交付承接：文章、模板、咨询、工具至少有 1 个。 [C]
4. 信任边界：不碰违法、欺骗、隐私、虚假收益。 [B]
5. 复盘转化，不只看曝光。 [C]

## ⑤ 反噬红线

- 黑红会损害长期信任资产，非必要不碰。 [C]
- 只有争议没有交付，等于消耗信用。 [C]
- 热点生命周期短，必须在 24–72 h 内沉淀资产。 [C]
- 规则未稳领域必须先写最坏后果和退出线。 [B]

## ⑥ if-then 卡

- if 追热点前无承接页 → 不发。 [C]
- if 钩子必须夸大才成立 → 删除。 [C]
- if 曝光高转化低 → 查信任层与交付层。 [C]
- if 争议伤害长期身份 → 放弃短期流量。 [C]
- if 弱共识窗口关闭 → 转入长期资产页，不再追。 [C]
