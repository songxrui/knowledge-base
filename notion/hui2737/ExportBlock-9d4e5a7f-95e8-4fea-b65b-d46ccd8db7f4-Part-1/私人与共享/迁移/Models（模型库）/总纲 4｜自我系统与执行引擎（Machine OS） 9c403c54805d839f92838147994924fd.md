## ① 一句话模型

稳定执行 = 带宽 × 能量 × MED × 环境防火墙 × 现实反馈。 [B]

## ② 五步机器诊断

1. Goal：我今天唯一目标是什么？ [B]
2. Problem：现在卡在哪里？事实，不写自责。 [B]
3. Diagnosis：根因是带宽、能量、恐惧、环境还是激励？ [B]
4. Design：改一条规则、环境或任务剂量。 [B]
5. Do：25 分钟内产生可见交付。 [B]

## ④ 三层修复优先级

- P0 带宽：手机隔离 + 25 分钟单块 + 一个可见文件。 [C]
- P1 能量：早光 10 分钟 + 步行/训练 + 慢呼吸。 [B]
- P2 防火墙：P0 前不刷信息流；睡前 30–60 分钟低刺激。 [C-self]

## ⑤ MED 修复卡

- 动作≤2 个；每个动作必须有剂量、频率、完成标准。 [C]
- 达标即停，不加码。 [C]
- 失败后只改一个变量；不重写整个人生。 [C]
- 7 天后用事实复盘，不写情绪散文。 [C]

## ⑥ if-then 卡

- if 起床就刷手机 → 手机移出卧室，起床先水+光。 [C-self]
- if 任务启动不了 → 降到 5 分钟物理动作。 [C]
- if 情绪 >7/10 → 走 10 分钟 + 双吸一呼×3。 [B]
- if 整理系统超过 30 min 无交付 → 停止整理，生成 1 个文件。 [C]
- if 连续 7 天 P0 为 0 → 环境隔离 + 任务降到最小交付。 [C]
