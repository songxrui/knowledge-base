**纳瓦尔｜杠杆自由引擎**：财富是能在你睡觉时工作的资产；自由来自低欲望、好判断、可复制杠杆与长期复利。

## ① 一句话模型

财富 = 专长 × 责任 × 杠杆 × 判断力 × 时间；目标是自由，不是地位。 [B]

## ② 三个目标别混

| 对象 | 定义 | 误区 | 动作 |
| --- | --- | --- | --- |
| 财富 | 能持续产生价值的资产 | 以为工资就是财富 | 建可复制资产 |
| 金钱 | 财富转移的欠条 | 只盯余额 | 换资产与自由度 |
| 地位 | 零和排序 | 为赢别人牺牲主线 | 退出地位游戏 |

## ③ 财富四变量

1. **专长**：社会需要但难以批量培训的能力；来自好奇心、长期问题和个人经历交叉。 [B]
2. **责任**：用自己名字承担结果；没有责任就没有超额回报。 [B]
3. **杠杆**：劳动 < 资本 < 代码/媒体；代码和媒体复制成本趋近 0。 [B]
4. **判断力**：杠杆越大，方向错误的代价越大；先选方向再努力。 [B]

## ④ 判断力协议

- 读基础学科，不迷信二手商业书：经济学、博弈论、心理学、进化论。 [B]
- 重大选择若势均力敌，优先选短期更难但长期更自由的一边。 [C]
- 保留空白时间；高密度输入后必须留散步/发呆窗口。 [C]
- 若一个方向不能积累资产、声誉或技能，默认不是主线。 [C]

## ⑤ 平静协议

- 欲望 = 与自己签订的“不满足合同”；同一时期只保留 1 个强欲望。 [C]
- 健康 > 心理稳定 > 关系 > 财富；上游崩，下游归零。 [B]
- 嫉妒急救：是否愿意 100% 交换对方人生？不愿则停止比较。 [C]
- 退休 = 不再为明天回报牺牲今天的全部。 [C]

## ⑥ if-then 卡

- if 收入仍与时间强绑定 → 每周至少 1 个可复制资产块。 [C]
- if 追热点偏离专长 → 回到“我长期好奇且能承担责任的问题”。 [C]
- if 想炫耀地位 → 问它是否增加自由度。 [C]
- if 输入很多但资产为 0 → 停读，输出一张卡或一篇短文。 [C]
- if 低欲望变成懒散 → 用 P0 交付物校验。 [C]
