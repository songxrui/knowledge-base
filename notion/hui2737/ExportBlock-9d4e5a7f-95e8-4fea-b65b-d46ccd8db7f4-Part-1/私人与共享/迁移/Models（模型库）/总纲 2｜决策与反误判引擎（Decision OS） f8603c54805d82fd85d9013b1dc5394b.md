## ① 一句话模型

好决策 = 目标清晰 × 世界判别 × 反误判 × 可信度加权 × 可修复。 [B]

## ② 8 步决策流程

1. 定题型：方向选择 / 执行优化 / 风险下注。 [B]
2. 写目标函数：长期自由、现金流、健康、声誉还是关系？ [B]
3. 判世界：平均世界可优化；极端世界先风控。 [A]
4. 列关键变量≤7：必须可观察或有代理指标。 [B]
5. 写出局线：健康、信用、现金、关系、时间的熔断点。 [A]
6. 可信度加权：成功≥3次且能解释机制者高权重。 [B]
7. 写 3 条反证：成立即不做。 [B]
8. 恐惧设定：最坏→预防→修复→不作为代价。 [C]

## ④ 可信度加权规则

- 一手数据 / 真实项目 / 可解释机制 → 高权重。 [B]
- 有经验但样本少 → 小规模试。 [C]
- 只会讲观点、卖焦虑、无代价 → 降权或过滤。 [B]
- AI 输出默认≤C；进入决策前必须核查。 [C]

## ⑤ if-then 卡

- if 情绪强度 >7/10 → 24 h 冷却再决策。 [B]
- if 最坏结果不可修复 → 不做或重构为小实验。 [A]
- if 只有支持证据 → 强制找 3 条反证。 [B]
- if 信息源无共担 → 只当观点，不当依据。 [B]
- if 变量超过 7 个 → 删除到少数关键变量。 [B]
