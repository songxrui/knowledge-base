Model Type: Investment

6) 周复盘纪律偏差
When to Use: 任何与资产配置、下注、交易、现金流、项目资源投入相关的决策。先保证不出局，再用价值/仓位/杠铃/数据纪律争取长期几何增长。

## ③ 统一流程

1. 判世界：能否单点归零？能 → 按极端世界处理。 [A]
2. 写出局线：什么情况必须停、撤、降仓？ [A]
3. 找 Edge：价值误定价或统计规律；无 Edge 不下注。 [B]
4. 定仓位：单次亏损≤可承受损失；现实用保守仓位。 [A]
5. 配杠铃：80–95% 保守底座 + 5–20% 期权端。 [B]
6. 周复盘：回撤、仓位、纪律偏差、熔断执行。 [B]

## ⑤ if-then 卡

- if 最坏结果不可承受 → 不做或砍到可承受规模。 [A]
- if 只因涨跌心动 → 先估价值；不估不买。 [B]
- if 无 Gate/Trigger/Sizing/Exit → 不交易。 [B]
- if 连续亏损还想加码 → 触发熔断，降仓或暂停。 [A]
- if 一个机会不能积累资产/技能/声誉 → 不做主线。 [C]
