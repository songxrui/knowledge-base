# 价值笔记｜最小总纲

<aside>
⚡

**价值笔记｜马斯克原理版**：先不出局，再创造有用功；先删损耗，再做积累；先到一线验证，再沉淀系统。所有笔记、计划、输入、关系和工具，只保留能改变行动、保护自由、生成反馈、制造资产的部分。

</aside>

## ① 一句话总纲

长期价值 = 不出局 × 有用功 × 清醒判断 × 一线反馈 × 可复用工厂；先删掉低效系统，再把注意力投入能复利的现实产出。 [C-Musk]

## ② 价值排序

| 顺位 | 保什么 | 删什么 |
| --- | --- | --- |
| 1 | 健康与睡眠 | 透支身体换短期兴奋、用极端工时证明价值 |
| 2 | 清醒与判断 | 叙事麻醉、情绪下注、权威崇拜、脑内规划 |
| 3 | 现金流与自由 | 一次出局的杠杆、债务依附、讨好型承诺 |
| 4 | 一线反馈 | PPT、空泛复盘、收藏、想象中的用户 |
| 5 | 作品与技能 | 输入成瘾、系统折腾、无产出的学习 |
| 6 | 可复用工厂 | 一次性爆发、靠意志维持的流程 |
| 7 | 高质量关系 | 羞辱、控制、单向索取、消耗型关系 |
| 8 | 足够财富 | 炫耀、身份消费、虚荣排名 |

地位不进核心排序。财富只在增加选择权、时间、健康、技能、作品和自由时有价值；若削弱健康、清醒、信誉或自由，降级处理。 [C]

## ③ 价值公式

**基础版**：长期价值 = 不出局 × 健康 × 清醒 × 真实反馈 × 可复利作品。 [B/C]

**马斯克加固版**：长期价值 = 不出局 × 有用功 × 一线反馈 × 可复用工厂 ÷ 白痴指数。 [C-Musk]

变量解释：

- **不出局**：健康、信誉、现金流、自由、关系网不被一次行动摧毁。
- **有用功**：帮助人数 × 平均帮助强度。
- **一线反馈**：来自代码、用户、身体、账单、作品、测试、真实行为的数据。
- **可复用工厂**：能重复生产结果的模板、流程、技能、产品、内容系统。
- **白痴指数**：总投入 / 核心增值投入；指数越高，越该删除或重构。

## ④ 七条不可再减原则

1. **先不出局**：会摧毁健康、信用、现金流、自由、关系网的事，一票否决。 [B]
2. **事实胜过叙事**：先写发生了什么，再写解释；没有证据的解释只算假设。 [B]
3. **有用功胜过虚荣**：先问“它帮谁、帮多少、改善多大”；说不清对象和效用，不进 Core。 [C-Musk]
4. **系统胜过意志**：好行为降门槛，坏行为加摩擦；不要把自己暴露给高刺激环境。 [B]
5. **删除胜过优化**：优化一个不该存在的东西，是最大浪费。 [C-Musk]
6. **一线胜过脑补**：没有现实反馈的计划，只是叙事。 [C-Musk]
7. **复利胜过刺激**：每天至少留下一个能被明天继续使用的产出物。 [C]

## ⑤ Core 笔记入库标准

| 问题 | 进 Core | 不进 Core |
| --- | --- | --- |
| 会改变行动吗？ | 未来 6 个月会改变决策/动作 | 只是“觉得有道理” |
| 能创造有用功吗？ | 明确帮助对象、改善幅度、使用场景 | 只是表达观点或证明自己聪明 |
| 能复用吗？ | 可跨 2 个以上场景调用 | 只服务一次情绪 |
| 能验证吗？ | 有指标、DoD、7 天实验、一线反馈 | 只有观点、语录、鸡汤 |
| 能压缩吗？ | 能写成 if-then / SOP / 模型卡 / Prompt | 必须靠长文维持存在感 |
| 能降低白痴指数吗？ | 能减少步骤、返工、切换、沟通、工具成本 | 新增复杂度但不减少损耗 |
| 能替代旧页吗？ | 合并后可归档 ≥1 个旧块 | 新增但不删除 |

规则：同一主题出现第 3 页时，禁止再新建；必须合并为 1 页。 [C]

## ⑥ 日常最小协议

- **每日唯一 P0**：留下一个可见文件或行动证据；没有证据不算做过。 [C-self]
- **P0 有用功**：写下今天 P0 帮谁、改善什么、产出什么。 [C-Musk]
- **高刺激三闸**：起床后 60 分钟、P0 前、睡前 60 分钟，不刷推荐流/游戏/看盘。 [C-self]
- **输入处理**：48 小时内处理 Inbox；不处理默认降级。 [C]
- **学习压缩**：3 洞察 + 2 行动 + 1 句费曼。 [C]
- **一线验证**：每天至少一次接触原始材料、代码、数据、身体指标、用户反馈或作品结果。 [C-Musk]
- **周复盘一问**：本周最大复利动作和最大无效投入各是什么？ [C]

## ⑦ 七个 if-then

- if 想做重大决定且情绪 >7/10 → 24 h 冷却 + 写决策纸。 [B]
- if 想保存新材料 → 先写“它会改变我什么行动”。写不出则不存。 [C]
- if 一页内容和旧页相似 ≥60% → 合并旧页，禁止并列。 [C]
- if P0 连续 7 天为 0 → 降到 5 分钟可见交付，先恢复现实反馈。 [C]
- if 关系持续羞辱、控制、单向索取 → 降级，不反复解释。 [C]
- if 想靠消费/游戏/信息流止痛 → 写触发源，走 10 分钟，再做 5 分钟 P0。 [C-self]
- if 笔记让你焦虑而不改变行动 → 归档或删除。 [C]

## ⑧ 马斯克原理 if-then

- if 听到“这行都这样” → 问：这违反物理、法律、现金流、身体约束吗？不违反则可重构。 [C-Musk]
- if 一个流程总耗时远大于核心增值时间 → 计算白痴指数，先删中间损耗。 [C-Musk]
- if 想优化流程 → 先问它是否该存在。 [C-Musk]
- if 想买工具或搭系统 → 先手动跑通 3 次，再决定是否自动化。 [C-Musk]
- if 判断卡住 → 到一线：看代码、看原文、看数据、问用户、做样品。 [C-Musk]
- if 一个目标看似不可能 → 拆成场地、资源、时间、技能、反馈、瓶颈，而不是直接放弃。 [C-Musk]
- if 想模仿马斯克极端强度 → 先写健康、现金流、关系、恢复能力的下行保护；写不出则禁止照搬。 [C-Musk]

## ⑨ 价值红线

| 红线 | 动作 | 理由 |
| --- | --- | --- |
| 长期睡眠透支 | 停止加任务，先修节律 | 上游崩，下游全崩 |
| 高杠杆负债/冲动下注 | 不做或砍到可承受规模 | 一次出局毁复利 |
| 涉钱/健康/合同/论文撞谎 | 沉默或讲实话，不编造 | 信誉是底层资本 |
| 高刺激接管注意力 | 物理隔离设备 + 信息断食 | 注意力是自由入口 |
| 只输入不输出 | 停止输入，生成一张卡 | 输入不转化就是负债 |
| 长期消耗型关系 | 降级或退出 | 关系会改写行为分布 |
| 用愿景包装压榨 | 停下，写边界、代价、恢复方案 | 使命不能覆盖健康和伦理 |
| 过早自动化 | 回到手动流程和五步算法 | 自动化会放大错误流程 |

## ⑩ 笔记库减法协议

1. **一主题一核心页**：相似内容只允许一个当前版本。 [C]
2. **旧页不删先归档**：保留来源，但不占前台入口。 [C]
3. **标题降级**：不是入口不叫 OS；不是闭环不叫系统；不是生成模块不叫引擎。 [C]
4. **页内先结论**：定位 → 规则 → 动作 → 例外 → 来源。 [C]
5. **每次整理必须减少前台页数**：只整理不归档，视为无效整理。 [C]
6. **新增页必须降低白痴指数**：不能减少未来检索、判断或行动成本的页面，不新建。 [C-Musk]
7. **笔记必须连接一线反馈**：没有原文、案例、数据、行动证据或复盘结果，只能进 Inbox，不能进 Core。 [C-Musk]

## ⑫ 读书笔记入库协议

读书笔记不是“看完一本书”，而是把书变成可调用的行动系统。 [C-Musk]

一本书进入 Core，必须至少产出：

1. 一句话总纲。
2. 7–12 个模型卡。
3. 5–10 个案例卡。
4. 10 条可执行原则。
5. 反误用清单。
6. 7 天实践方案。
7. 一个下次调用 Prompt。
8. 至少 1 个现实流程被修改。

**读书笔记白痴指数** = 阅读总时间 / 产生可复用模型、行动和系统的时间。指数 >3，说明读得太散。 [C-Musk]

## ⑬ AI / Codex / Skill 价值协议

- AI 用于压缩、重构、对照、审稿、生成模板，不直接替代现实判断。 [C]
- Codex 用于读代码、跑测试、生成补丁、复盘错误，不用于逃避理解。 [C]
- Skill 只沉淀高频、稳定、必要、可验证流程。 [C-Musk]
- 自动化前必须完成：质疑需求、删除、简化、手动跑通 3 次。 [C-Musk]
- 每个 AI 工作流必须有验证方式：测试、人工审稿、样例对比、输出复用率、错误率。 [C]

## ⑭ 反误用清单

| 想学的东西 | 正确学法 | 错误学法 |
| --- | --- | --- |
| 第一性原理 | 拆事实、算成本、做实验 | 凭自信否定专家 |
| 硬核工作 | 短期冲刺、长期恢复 | 长期透支身体 |
| 风险偏好 | 小注试错、逐步加码 | 无现金流裸赌 |
| 反馈直接 | 批评事实和行为 | 羞辱人格 |
| 愿景使命 | 提供方向和筛选标准 | 包装压榨和逃避现实 |
| 自动化 | 最后一步 | 一开始就搭复杂系统 |
| 一线原则 | 接触真实反馈 | 什么都亲自插手 |
| 速度 | 缩短反馈周期 | 鲁莽牺牲质量和健康 |

## ⑯ 证据等级

- [A] 一手研究 / Meta 分析。
- [B] 权威综述 / 教材 / 指南。
- [C] 长期实践者经验。
- [C-self] 本人经验。
- [C-Musk] 来自《马斯克原理》读书笔记提炼的模型化实践，属于可迁移推断，使用时需保留健康、现金流、法律、关系边界。
- [D] 二手解读，仅作线索。

低于 [C] 的内容不进入总纲；[C-Musk] 不等于照搬马斯克，只吸收第一性拆解、删减、一线反馈、制造结果、速度反馈和反熵增系统。
