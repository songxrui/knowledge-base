## 关键句子翻译

### "这不是优化问题，是信任问题。"
→ "This is not an optimization problem. It is a trust problem."

### "草稿不能叫产品。"
→ "A draft is not a product."

### "选错一个skill，比不用更差。"
→ "Picking the wrong skill is worse than using none at all."

### "7种文件解决的是信任问题，不是功能问题。"
→ "The seven file types solve a trust problem, not a functionality problem."

### "先消灭隐身税，再让市场决定。"
→ "First, eliminate the invisibility tax. Then, let the marketplace decide."
