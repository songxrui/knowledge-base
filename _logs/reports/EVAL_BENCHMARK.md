# EVAL_HARNESS_BENCHMARK — 35卡评测基准

> 基于 eval-harness skill 原则建立 | 用于自动化评测每张卡片质量

## 评测维度（10维·各0-1分·满分10）

| # | 维度 | 0分标准 | 1分标准 | 评测方式 |
|---|------|--------|--------|---------|
| D1 | 证据密度 | 0条引用或全伪引用 | ≥2条真实来源·出处可查 | grep来源字段 |
| D2 | 深度推论 | 仅复述原文 | 有成立前提+边界+反例+第二层推论 | 结构完整性检查 |
| D3 | 反共识角度 | 无 | 有独立反共识段落·与主流说法对比 | grep反共识字段 |
| D4 | 优化日志 | 无 | 有五轮优化日志表 | grep优化日志字段 |
| D5 | 横向连接 | 0条 | ≥3条·含具体卡ID | grep相关条目字段 |
| D6 | 可执行性 | 动作模糊无步骤 | ≥2条·有数字/时间/验收标准 | grep可执行动作用语 |
| D7 | 个人数据绑定 | 无用户真实数据 | 含体重/亏5300/游戏排名等具体数据 | 关键词匹配 |
| D8 | 一句话洞察 | 无或为废话 | 一句话·敢下判断·可记忆 | 格式检查 |
| D9 | 禁用词 | 出现黑名单词 | 全扫清 | regex扫描 |
| D10 | 密度/字数 | <3KB或注水 | 4-7KB·信息密度高 | 字数统计 |

## 自动化评测命令

```powershell
# D1 证据密度
Select-String -Path cards\C*-*.md -Pattern '证据\d' | Measure-Object | Select -Expand Count

# D4 优化日志
Select-String -Path cards\C*-*.md -Pattern '优化日志' | Measure-Object | Select -Expand Count

# D9 禁用词扫描
$banned=@("赋能","抓手","闭环","底层逻辑","本质上","综上所述","众所周知"); Select-String -Path cards\C*-*.md -Pattern ($banned -join '|')
```
