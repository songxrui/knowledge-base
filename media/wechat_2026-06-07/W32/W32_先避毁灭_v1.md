# 二十二岁的人最该学的一课不是怎么成功——是怎么确保自己不会永久出局

芒格说过一句话，我把它翻译成了二十二岁的版本。

他说，如果我知道我会死在哪里，我就不去那里。

二十二岁的翻译版：如果你知道什么东西能让你的人生永久崩盘，提前绕开它。

不是不冒险。是只冒可逆的险。

亏五千三百块是可逆的。但用母亲的钱亏的——把可逆的亏损变成了半不可逆的信任违约。这就是不可逆错误的教科书级案例。

（来源：C2-1_先避毁灭再求收益.md + Notion·交易系统复盘）

二十二岁最大的财富是你可以犯一百个错误而每一次的成本接近零。但这个特权有一个精确的前提——你犯的是可逆的错误。不可逆的错误只有四种。健康毁灭、信誉毁灭、自由毁灭、关键关系毁灭。

健康毁灭的意思不是你偶尔熬夜。是长期每天睡不足六小时——你的大脑不会后悔了重新长一个。信誉毁灭的意思不是有人说你坏话。是你用别人的钱冒险被发现——信任修复成本等于金钱损失的十倍到一百倍。自由毁灭的意思很直白。违法犯罪。关键关系毁灭——你不在乎的人可以不在乎，但你在乎的人如果因为你的背叛而离开，修复周期以年为单位。

这四种毁灭之外的任何失败——亏钱、考试挂科、创业失败、被拒绝、文章没人看——都是可逆的。可逆的失败是学费。不可逆的失败是永久出局。

所以我给自己定了三条铁律。不用不属于自己的资本下注。不再有正期望值系统的时候投真金白银。每次做重要决定前问自己——最坏结果是什么，我能承受吗，我在替别人做决定吗。三条全是绿灯才动手。有一条黄灯就等。有一条红灯就撤。

这不是保守。是保证自己永远在场。而只要在场，你就有无限次翻盘的机会。

（来源：C3-2_杠铃策略.md + C7-3_反脆弱.md）

二十二岁最容易被忽略的真相——你最大的资产不是你的能力，是你未来五十年的收入折现值。五十年的价值取决于一件事——你有没有把自己搞到永久退场。只要没有，你的未来就值很多钱。只要搞了——一切归零。

先避毁灭。永远在场。其他的交给复利。
