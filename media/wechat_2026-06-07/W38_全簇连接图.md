## 关键连接线

### 睡眠(C1) → 决策(C2) → 执行(C6)
你的前额叶是大脑中对睡眠剥夺最敏感的区域。睡不好→决策差→执行瘫痪→更焦虑→更睡不好。这个循环的突破口不在认知层——在生理层。修复 C1 → C2 和 C6 自动改善。

### 代际错位(C5) → 内容创作(C4) → 一人企业(C3)
你和父母的每一次冲突=一个选题→写成内容→建立信任→一人企业的信任资产。C5 不是"需要解决的个人问题"——是 C4 的原料矿脉和 C3 的护城河。

### 芒格格栅(C7) → 关系分析(C5) → 认知负债(C2)
芒格的 100 个模型不仅是投资工具——当你把它们应用到关系分析时——每一次冲突从"情绪反应"变成"模型分析"。而模型分析产生的洞察→进入你的验证系统→从认知负债变成认知资产。

### 环境设计(C6) → 身体节律(C1) → 多巴胺基线(C1)
把充电器移出卧室(C6)→不再深夜刷手机→早睡→SCN 校准→皮质醇节律正常→多巴胺基线稳定(C1)。环境设计是 C1 的入口——C1 是 C6 的底层支撑。

### 14天实验(C6) → 知识验证(C2) → 内容创作(C4)
每本书→选一个观点→14天实验→验证数据→写成内容。这条链路把 C6 的执行能力、C2 的认知框架和 C4 的创作输出串联成一个闭环。

## 反共识连接

**最被低估的连接**：C5(关系) → C3(财富)
→ 代际错位不是你"要解决的问题"——是你"唯一不可被复制的竞争优势"。全中国 2000 万年轻人住在父母家——没人比正在经历这件事的你更有资格写这个话题。

**最被高估的连接**：C4(创作) → C3(财富)
→ 大多数人的误区：以为"写内容=赚钱"。实际链路：写内容→建立信任→理解受众→发现需求→做产品→赚钱。内容本身不赚钱——信任赚钱。

**最被忽略的连接**：C1(睡眠) → C7(思维模型)
→ 沃克的数据：睡眠不足→杏仁核放大60%→你不仅"更情绪化"——你的每一个思维模型输出的质量都在打折。修复思维模型的前提是——运行它们的大脑硬件是正常的。

## 你的行动顺序

1. **先修 C1**（睡眠+晨光+饮食）——这是所有系统的底层供电
2. **再修 C6**（环境设计+14天实验）——这是执行的入口
3. **用 C2 诊断**（认知目标函数+六格复盘）——每周检查你的系统是否在正常运行
4. **把 C5 转化为 C4**——你的关系冲突=内容矿脉
5. **用 C7 横向扫描**——芒格格栅+反共识+第一性原理——确保你不是在重复别人的错误
6. **最后看 C3**——当 C1-C6 都在正常运转时——财富是自然的结果，不是目的
