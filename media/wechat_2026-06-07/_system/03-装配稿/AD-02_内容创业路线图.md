# AD-02｜22岁从0到第一块钱的内容创业路线图（装配稿）

**类型**：选题装配稿
**目标受众**：22-25岁零资本、零团队、想做独立创作的年轻人
**核心问题**：不借钱、不辞职、不住北上广——怎么从0赚到第一块钱？

## 装配结构

### 开头(Hook)
- CAS-02：5300元亏损·用母亲的钱→三问全红
- QST-02：没资源怎么开始？

### 核心论点
- OPI-02：22岁最大资产不是存款·是"还能再押100次"
- CON-02：一人企业模型·Dan Koe·Social posts=MVP
- 支持数据：董辉93篇媒体稿·12篇后3人私信

### 解决方案
- SOL-02：4阶段时间表(30/60/90天)
- 安全端设计：住父母家压低成本·用时间+创意替代金钱冒险
- 注意力配置：60%创作·25%学习·15%消费(Cal Newport)

### 收尾
- 一句话：先让一个人值得被关注和信任——10篇公开发布的内容比10页商业计划书更接近这个目标。

## 调用单元
QST-02 + CON-02 + OPI-02 + CAS-02 + SOL-02
