# 30天读书行动计划 — 如何把这本书变成你的操作系统

> 本书~137,000字、~117本名著。如果你只读一遍，你会得到知识。如果你用30天按以下计划执行，你会得到一套新的操作系统。

## 使用原则

1. **每天只读一章的一节**（约20-30分钟），不做"快速读完"——这不是小说
2. **读完立刻做一个动作**（每节末尾都有"落地"操作），不做"收藏以后用"
3. **用CH06的决策框架检查你的每日选择**——Musk五步 + Mungel清单 + 贝叶斯更新
4. **每周日做一次"周复盘"**——用CH07的复盘模板检查这一周

## 如果30天太长

- **7天速成版**：每天读一章的"引言"+"落地"部分（跳过深挖和反驳复审）
- **单章深读**：选和你当前最相关的1章，用1周读完+全部执行
- **随手翻**：把CH08的9模型决策矩阵打印贴墙上，每次做决策前扫一眼

> 这本书的真正价值不是你"读完"它，而是你"做完"它。137,000字的知识在你大脑里留存率约10%——但是30个"做了的动作"在你身体里留存率是100%。
