### 2.2 单元文件命名规范

格式：`{TYPE}-{YYYYMMDD}-{SEQ}_{原文标题摘要}.md`

示例：`SOL-20260611-001_D01 AI时代独立思考价值的三步方案.md`

| 字段 | 说明 |
|------|------|
| TYPE | QST/CON/OPI/CAS/SOL |
| YYYYMMDD | 创建日期（均为2026-06-11） |
| SEQ | 3位序号 (001-311) |
| 标题 | 原文标题或核心内容摘要 |

## 五、使用指引

### 内容创作工作流

```
输入（灵感/素材）
    ↓
[01-原始素材区] 存放原始素材（W系列/D系列）
    ↓
[02-内容单元库] 拆解为 QST/CON/OPI/CAS/SOL 单元
    ↓
[05-主题地图] 按主题组织相关内容单元
    ↓
[06-选题装配] 装配成可发布内容骨架
    ↓
[01_Projects] 撰写完整长文
    ↓
[media/] 平台适配分发
```
