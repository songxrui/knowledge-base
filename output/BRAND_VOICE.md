# 董辉内容品牌语调档案

> Skill: brand-voice | 来源：35张深度卡 + 3篇装配稿 + 决策原则文档
> 用途：后续所有内容创作统一语调·跨平台适配保持一致性

---

## 一、语调核心定位

**一句话定位**：22岁·长沙·AI一人产品探索者·系统崩溃者自救记录者·代际错位叙事者

**语调三元**：
- **智识但不学术**：有理论框架（多巴胺/第一性原理/芒格格栅）但不用学术黑话
- **诚实但不自怜**：暴露系统崩溃/游戏成瘾/亏5300/家庭冲突，但不制造受害者叙事
- **可操作但不鸡汤**：每篇有数字/步骤/验收标准，没有"你一定可以的"空话

---

## 二、语调特征提取

### 节奏与句长
- 句子短但不断裂（15-25字为主，偶尔40字长句用于深度推演）
- 段落短（2-4句/段），适合移动端阅读
- 每段结尾有"重击"——一个尖锐的判断或一个具体数字
- 避免：长排比、修辞堆砌、过度分段

### 压缩度
- 高压缩比——能用5个字不写10个字
- 例："你不是懒"（4字）替代"你并不是缺乏自律能力"（9字）
- 例："睡眠不是休息"（5字）替代"睡眠不仅仅是身体休息的时间"（11字）
- 禁用词："值得注意的是""综上所述""随着…的发展"

### 证据感
- 每150-200字至少1个具体数字/案例/出处
- 数字优先于形容词——"每天屏幕时间10小时"而非"手机使用过多"
- 个人数据优先于通用数据——"我亏了5300"而非"许多年轻人投资亏损"
- 引用规则：微读/Notion/飞书出处可追溯

### 观点锐度
- 敢于下判断——"完美主义是创作者最大的敌人"而非"完美主义可能对创作有不良影响"
- 每个核心段落有一句"敢下判断的核心论点"
- 但判断后紧跟依据——"因为…"而非"我觉得…"
- 避免："可能""或许""某种程度上"的过度稀释

### 游戏化语言
- 大量使用游戏类比（来源：10年王者荣耀经历）
- 例："杠铃策略=坦克出装""P0完成率=经验值条""六格复盘=看回放找失误"
- 游戏类比作为"降低认知门槛"的工具，不是"显得自己会玩游戏"

### 情绪基调
- 冷静但不冷漠——分析情绪但不被情绪裹挟
- 幽默但不轻浮——自嘲（"我亏了5300买的教训"）而非嘲笑他人
- 有情感锚点——代际错位的无力感/系统崩溃的恐惧/创作的焦虑——这些情感被承认但不被放大

---

## 三、绝对不做（品牌红线）

| 禁止行为 | 原因 |
|----------|------|
| 以"亲爱的读者""大家好"开头 | 不制造虚假亲密感 |
| 以"如果你觉得有帮助请点赞转发"结尾 | 不制造乞讨感 |
| 使用感叹号超过1个/段 | 不制造假兴奋 |
| "相信你自己""你可以的"类鸡汤 | 不给出无依据的鼓励 |
| 过度使用第一人称"我" | 平衡"我""你""我们" |
| 贬低他人来抬高自己 | 不制造对立 |
| 假谦虚（"我只是一个…"） | 诚实定位 |
| 编造自己没有的经历 | 证据规则第一位 |

## 四、平台语调适配

### 即刻（短内容·同行者心态）
- 更压缩：≤300字·1个观点·0废话
- 更直接："你不是懒。你是…"开门见山
- 更多#标签：用于社区发现
- 更多问句：引发讨论而非给出全部答案

### 小红书（种草·探索者心态）
- 更多视觉化：emoji适度·分段清晰·关键词加粗
- 更多"你"：面向读者的问题和需求
- 更多步骤化：1→2→3→4 清晰的操作路径
- 更温和：在即刻的锐度上降低10%

### 公众号（深度·追随者心态）
- 更长叙事：3000-5000字完整论证
- 更多交叉引用：引用自己的旧文章/卡片
- 更多理论+案例：先讲框架再讲个人故事
- 保持锐度：不和稀泥，敢下判断

---

## 五、语调自检清单（每次发布前）

- [ ] 开头第一句是否在10字内抓住注意力？
- [ ] 150字内是否出现了至少1个具体数字/案例？
- [ ] 是否有至少1句"敢下判断的核心论点"？
- [ ] 是否有禁用词（值得注意的是/综上所述/首先其次最后）？
- [ ] 结尾是否干净（无CTAs乞讨/无开放式废话）？
- [ ] 是否用了至少1个游戏/技术类比降低认知门槛？
- [ ] 是否符合"智识但不学术·诚实但不自怜·可操作但不鸡汤"三元？
